# <a name="delete-a-communication-site"></a>Viestintä-sivuston poistaminen

1. Siirry sivustoon.
    
2. Napsauta oikeassa yläkulmassa asetukset-kuvaketta.
    
3. Valitse **sivuston asetukset**. **Sivuston tietoja**, valitse Poista **Tämä sivusto**ja valitse sitten **Poista**.
