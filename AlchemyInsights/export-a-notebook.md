# <a name="export-a-notebook"></a>Muistikirjan vieminen

OneNote 2016 -työpöytäsovelluksessa:

1. Valitse vietävällä sivulla Tiedosto **> Vie**.
2. Valitse **Vie nykyinen**-kohdasta vietävä t. sivu, osa tai muistikirja.
3. Valitse **Vie**.
 
Jos haluat siirtää pilveen tallennetun tiedoston omistajuuden, voit siirtää muistikirjojen sisällön myös **Jaa-** tai **Kopioi-osan** ominaisuuksien avulla.  

- Valitse sovellusikkunan oikeasta yläkulmasta **Jaa**.
- Lisätietoja muistikirjan [siirtämisestä on ohjeaiheessa Muiden kanssa jakamasi OneNote for Windows -muistikirjan siirtäminen](https://support.office.com/article/move-a-onenote-for-windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4?ui=en-US&rs=en-US&ad=US).
