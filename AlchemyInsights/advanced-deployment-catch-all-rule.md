# <a name="deploy-office-apps"></a>Office-sovellukset käyttöön

Office-sovellusten käyttöönoton useiden käyttäjien kautta Office Deployment Tool, System Center Configuration Manager osana käyttöjärjestelmän levyn näköistiedoston tai jaetun tietokoneen skenaarion käyttäminen Windows Server Remote Desktop Services (RDS) tai Terminal Palvelut, harkita vaiheittainen opas- **Office 365 ProPlus käyttöönoton advisor**.
  
Aloittamiseen, varmista, että olet kirjautunut palveltavien tai Yleinen järjestelmänvalvoja ja napsauttamalla tätä: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
