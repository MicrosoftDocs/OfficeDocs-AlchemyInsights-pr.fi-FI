# <a name="use-sensitivity-labels-in-microsoft-365"></a>Luottamuksellisuusmerkkien käyttö Microsoft 365:ssä

Luottamuksellisuusotsikoiden avulla voit:
- Ota käyttöön suojausasetukset, kuten salaus tai vesileimat, osoitetussa sisällössä.
- Suojaa sisältöä Office-sovelluksissa eri käyttöympäristöissä ja laitteissa.
- Suojaa sisältöä kolmannen osapuolen sovelluksissa ja palveluissa Käyttämällä Microsoft Cloud App Securitya.
- Laajenna luottamuksellisuusmerkinnät kolmannen osapuolen sovelluksiin ja palveluihin.
- Luokittele sisältöä ilman suojausasetuksia.

Lisätietoja on kohdassa [Lisätietoja luottamuksellisuusmerkinnöistä.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
