# <a name="learn-about-microsoft-365-groups"></a>Lisä tietoja Microsoft 365-ryhmistä

Microsoft 365-ryhmät on palvelu, joka toimii jo käytössä olevien Office 365-työkalujen kanssa, joten voit tehdä yhteistyötä ryhmäsi kanssa, kun kirjoitat asia kirjoja, luot laskenta taulukoita, työskentelet projekti suunnitelmien parissa, kokoat koko uksia tai lähetät sähkö postia. Lisä tietoja on artikkelissa tietoja [Microsoft 365-ryhmistä](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
