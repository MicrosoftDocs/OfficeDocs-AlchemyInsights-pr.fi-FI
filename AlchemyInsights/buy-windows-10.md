# <a name="how-to-buy-a-copy-of-windows-10"></a>Kopion ostaminen Windows 10

Saat ajan tasalla hinnat ja ohjeet Windows 10:n täyden version ostamista ja [Microsoft Store.](https://www.microsoft.com/store/b/windows)

**Huomautus:** Maksuton Windows 10 Windows 10:n hankinta (GWX) -sovelluksen kautta päättyi 29. heinäkuuta 2016.

**Tärkeää:** Jos käytössäsi on Windows 7, Microsoft suosittelee, että siirryt uuteen Windows 10-tietokoneeseen, koska Windows 7:n tuen elinkaari päättyy 14. tammikuuta 2020.
