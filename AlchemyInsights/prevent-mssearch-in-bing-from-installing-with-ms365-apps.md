# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>Microsoft Searchin asennuksen estäminen Bingissä Microsoft 365 -sovellusten kanssa

Jos haluat estää Microsoft Search in Bingin taustapalvelun asentamisen Microsoft 365 Apps for Enterprisen kanssa, valitse jokin seuraavista vaihtoehdoista:

- Officen käyttöönottotyökalu tai ryhmäkäytäntö
- Microsoft Endpoint Configuration Manager (nykyinen haara) tai Microsoft Intune

Lisätietoja on kohdassa Microsoft Search in Bingin taustapalvelun asennuksen estäminen [Microsoft 365 Apps for Enterprisen kanssa.](https://go.microsoft.com/fwlink/?linkid=2151946)
