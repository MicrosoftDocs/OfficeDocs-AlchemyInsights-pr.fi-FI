# <a name="set-up-spf"></a>SPF:n määrittäminen

Päivitä DNS-tietueet niin, että voit käyttää lähettäjän käytäntökehystä (SPF) mukautetun toimialueen kanssa mukautetun toimialueen kanssa mukautetun toimialueen lähtevien sähköpostiviestien vahvistamiseen. Tämä auttaa estämään muita väärentämästä sähköpostia ikään kuin se olisi lähetetty toimialueestasi.
  
- [SPF:n määrittäminen estämään väärentäminen](https://docs.microsoft.com/microsoft-365/security/office-365-security/set-up-spf-in-office-365-to-help-prevent-spoofing)

- [Kaikkien DNS-tietueiden määrittäminen](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
