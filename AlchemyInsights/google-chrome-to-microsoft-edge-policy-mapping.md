# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Lisä tietoja Google Chromen käytäntöjen ja Microsoft Edge-käytäntöjen välisestä vastaamisesta

Google Chrome-käytännöt yhdistetään asiaankuuluviin Microsoft Edge-käytäntöihin, joita tuetaan selaimen versiossa 80. Lisä tietoja on artikkelissa [Google Chromen Microsoft Edge-käytäntöjen kartoitus](https://go.microsoft.com/fwlink/?linkid=2141933).
