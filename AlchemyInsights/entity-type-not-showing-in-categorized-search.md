# <a name="entity-type-not-showing-in-search-results"></a>Kohde tyyppi ei näy haku tuloksissa

Jos kohde tyyppi näkyy luokiteltu haku-tai osuvuus haku-kohdassa Dynamics 365 Unified Interface-liittymässä, se on ensin lisättävä sovellus moduuliin.

Huomautus: kun haet liitteitä osuvuus haussa, liite-entiteetti on lisättävä sovellus moduuliin, vaikka päätietue näkyy tuloksissa.
