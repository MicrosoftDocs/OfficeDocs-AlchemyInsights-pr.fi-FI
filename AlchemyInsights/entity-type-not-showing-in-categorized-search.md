# <a name="entity-type-not-showing-in-search-results"></a>Kohteen tyyppi ei näy hakutuloksissa

Entiteettityyppi luokitellut Dynamics 365 yhtenäinen käyttöliittymä Etsi tai merkitystä haun näkyviin se on ensin lisättävä app-moduuliin.

Huomautus: Kun etsitään liitteiden haun asiayhteyden, liite yksikön on lisättävä app-moduulin vaikka päätietue on mitä näkyy tuloksissa.
