# <a name="enable-seamless-sso"></a>Ota käyttöön saumattoman SSO

Azure Active Directory (Azure AD) saumaton kertakirjauksen (saumaton SSO) automaattisesti kirjautuu sisään käyttäjille kun ne ovat heidän yksittäisiä tietokoneita, jotka on yhdistetty yrityksen verkkoon.
  
[Noudata tämän artikkelin käyttöön saumattomien Kertakirjauksen](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
