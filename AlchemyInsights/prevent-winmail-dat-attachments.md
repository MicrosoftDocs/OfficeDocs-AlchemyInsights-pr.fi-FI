# <a name="help-prevent-winmaildat-attachments-in-email-messages-from-your-organization"></a>Winmail.dat-liitteitä organisaatiolta sähköposti viestien estäminen

Järjestelmänvalvojana toimi seuraavasti:

1. Avaa [Exchange-hallintakeskukseen](https://outlook.office365.com/ecp/).

2. Siirry **postin kulku** > **Remote toimialueet**.

3. Valitse oletus etätoimialueen nimeltä **Oletus**ja valitse sitten **Muokkaa**.

4. **Käytä RTF - muotoilu** -osan Valitse **ei koskaan**.

Lisätietoja [Määritä etä toimialueiden muotoa](https://docs.microsoft.com/Exchange/mail-flow-best-practices/remote-domains/remote-domains#specifying-message-format).
