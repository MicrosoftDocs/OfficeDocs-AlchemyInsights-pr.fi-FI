# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Ulkoiset käyttäjät eivät voi lähettää sähköpostia Microsoft 365 -ryhmälle

Näiden ohjeiden avulla ulkoiset käyttäjät voivat lähettää Microsoft 365 -ryhmälle sähköpostia:

1. Kirjaudu [hallintakeskuksen](https://admin.microsoft.com/)  >  **ryhmät**  >  **-ryhmiin**.
2. Avaa Asetukset-ruutu valitsemalla ryhmä, jota haluat hallita.
3. Valitse **Asetukset-välilehdessä** **Salli ulkoisten lähettäjien lähettää tämä ryhmä sähköpostitse**.
4. Valitse **Tallenna**.</br>
    Salli asetusten näkyminen enintään tunnin ajan.
