# <a name="how-to-install-mac-office-from-the-portal"></a>Mac Officen asentaminen portaalista

1. [Poista Office 2011 for Macin asennus tieto koneistasi](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Hallitse ohjelmiston lataus asetuksia](https://docs.microsoft.com/DeployOffice/manage-software-download-settings-office-365).

3. [Käyttö oikeuksien määrittäminen käyttäjille](https://docs.microsoft.com/microsoft-365/admin/subscriptions-and-billing/assign-licenses-to-users)

4. Käyttäjät voivat asentaa Office-ohjelmiston itse, tai voit käyttää [Officen käyttöönottoa Mac-käyttäjille](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac).
