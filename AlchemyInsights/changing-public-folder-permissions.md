# <a name="changing-public-folder-permissions"></a>Yleisen kansion käyttöoikeuksien muuttaminen

Sinä ja muut käyttäjät voi muuttaa yleisen kansion Outlookissa. Katso [määrittää tai muuttaa yleisen kansion käyttöoikeuksien](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Järjestelmänvalvojat muuttaa myös Keskikaiuttimen admin oikeudet:
  
- Microsoft 365 hallintakeskukseen, siirry **Admin keskittää** \> **Exchange**.
    
- Valitse **Yleiset kansiot**.
    
Sieltä voit muuttaa yksittäisten julkisten kansioiden käyttöoikeudet määrittämällä käyttäjiä, suojausryhmiä käyttöoikeudet. Voit muuttaa yleisen kansion käyttöoikeuksia, sinun on sen omistaja.
