# <a name="send-email-from-an-alias-or-secondary-address"></a>Sähköpostin lähettäminen aliaksesta tai toissijaisesta osoitteesta

Microsoft 365 ei tällä hetkellä tue sähköpostin lähettämistä aliaksesta tai toissijaisesta sähköpostiosoitteesta. Sähköpostialiaksien avulla voidaan vastaanottaa vain sähköposteja. Jos haluat lähettää sähköpostia toisesta sähköpostiosoitteesta, sinun on luotava jaettu postilaatikko. Jaettu postilaatikko antaa sinulle toisen sähköpostiosoitteen ilman toista käyttöoikeutta.
  
- [Luo jaettu postilaatikko tästä](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

Voit myös lukea [Luo jaettu postilaatikko -artikkelin,](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox) jossa kerrotaan kaikki, mitä sinun tarvitsee tietää jaetuista postilaatikoista, mukaan lukien niiden käyttäminen.
