# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Käyttäjän postilaatikon muuntaminen jaetuksi postilaatikoksi

Kun muunnat yksittäisen käyttäjän postilaatikon jaetuksi postilaatikoksi, kaikki aiemmin luodut sähköposti- ja kalenterikohteet säilytetään jaetussa postilaatikossa, johon useat käyttäjät voivat käyttää. Voit myös muuntaa jaetun postilaatikon takaisin käyttäjän (yksityisen) postilaatikoksi.
  
Lisätietoja on [ohjeaiheessa Käyttäjän postilaatikon muuntaminen jaetuksi postilaatikoksi](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
