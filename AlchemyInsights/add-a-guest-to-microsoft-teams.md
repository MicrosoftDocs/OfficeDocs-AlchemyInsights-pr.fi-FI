# <a name="add-a-guest-to-microsoft-teams"></a>Lisää vieras Microsoft Teamsiin

Joskus haluat tehdä yhteistyötä sellaisten ihmisten kanssa, jotka eivät ole työryhmän organisaatiossa. Usein teet sen sähköpostitse, mutta jos haluat tehdä yhteistyötä heidän kanssaan Teamsissa, voit lisätä heidät vieraina.

Lisä tietoja on kohdassa [Lisää vieraita tiimiin tiimissä](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop).

**Huom:** Sinun on oltava tiimin omistaja, jotta voit lisätä vieras Teamsiin (Katso myös [vieras-oikeuksien asettaminen](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)).
