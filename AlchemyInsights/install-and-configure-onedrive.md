# <a name="install-and-configure-onedrive"></a>OneDriven asentaminen ja määrittäminen

OneDrive-synkronointiohjelma on esiasennettu Windows 10:n kanssa. Jos haluat asentaa OneDriven Windows 7:lle tai Macille tai hankkia [https://onedrive.com/download](https://onedrive.com/download)puhelimen tai tabletin mobiilisovelluksen, siirry kohtaan .
  
Lisätietoja OneDrive-synkronointiohjelman määrittämisestä Windowsissa on seuraavissa ohjeissa:
  
- [Tietokoneen määrittäminen synkronoimaan OneDrive for Business -tiedostot](https://go.microsoft.com/fwlink/?linkid=533375)

- [SharePoint-tiedostojen synkronointi OneDrive-synkronointisovelluksen avulla](https://go.microsoft.com/fwlink/?linkid=871666)
