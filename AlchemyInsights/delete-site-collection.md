# <a name="delete-a-sharepoint-site"></a>SharePoint-sivuston poistaminen

1. Valitse uuden SharePointin hallinta keskuksen sivustot-sivulla sen sivuston vieressä oleva valinta ruutu, jonka haluat poistaa.
    
2. Valitse valinta nauhan työkalu riviltä **Poista**.
