# <a name="configure-privilege-for-user-and-system-chart"></a>Käyttöoikeuksien määrittäminen käyttäjä- ja järjestelmäkaaviota varten

**Ongelma:** Kaavion lukuoikeutta ei ole.

**Syy:** Ei tarpeeksi oikeuksia tai ei sisällä tarvittavia käyttöoikeusvirheitä.

**Ratkaisu:** Määritä kaavion oikeus

**Vaiheet:**

1. Valitse Asetukset **-> mukauttaminen -> -> käyttöoikeusroolit.**

2. Kaksoisnapsauta roolia.

    - Käyttäjäkaavio-oikeusmääritys on **Perustietueet-välilehdessä.**
    - Järjestelmäkaavio-oikeusmääritys on **Mukauttaminen-välilehdessä.**
