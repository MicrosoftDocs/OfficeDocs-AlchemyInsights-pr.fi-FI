# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive on juuttunut muutosten käsittelemiseen

Jos OneDrive näyttää "käsittely muutokset" pitkäksi aikaa, se voi johtua siitä, että online-tiedosto on avattuna, jonossa on paljon tiedostoja tai tiedoston synkronointi on hyvin suuri. Lue  [OneDrive on juuttunut "käsittely muutokset"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) ehdotettuja vaiheita ja lisä tietoja.

Jos tarvitset lisä ohjeita, voit  [lähettää OneDrive-tukea sähköpostitse](https://go.microsoft.com/fwlink/p/?LinkId=528676).
