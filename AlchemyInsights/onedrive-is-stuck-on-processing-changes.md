# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive on jumissa muutosten käsittelyssä

Jos OneDrive näyttää "Muutosten käsittely" -kohdassa pitkään, se voi johtua siitä, että sinulla on online-tiedosto avoinna, paljon jonoon juuttua tiedostoja tai erittäin suuri tiedostojen synkronointi. Lue [OneDrive on jumissa kohdassa "Muutosten käsitteleminen"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) ehdotettuja vaiheita ja lisätietoja varten.

Jos tarvitset lisää apua, voit [lähettää sähköpostia OneDrive-tukeen](https://go.microsoft.com/fwlink/p/?LinkId=528676).
