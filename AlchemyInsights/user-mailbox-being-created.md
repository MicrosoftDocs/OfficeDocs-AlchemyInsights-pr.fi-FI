# <a name="your-users-mailbox-is-being-created"></a>Käyttäjän postilaatikko luodaan

Se voi kestää jopa tunnin ennen kuin uusi postilaatikko on käyttövalmis.
  
Jos käyttäjän postilaatikkoon on ollut ”varautuminen” yli tunti, voit kokeilla poistetaan ja luodaan uudelleen käyttäjän tai ”soittaa sinulle kerro meille”-vaihtoehdon avulla voit puhua tukihenkilö.
