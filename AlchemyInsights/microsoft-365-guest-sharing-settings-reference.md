# <a name="microsoft-365-guest-sharing-settings-reference"></a>Microsoft 365 -vieraskäyttäjien asetukset

Seuraavassa artikkelissa on viittauksia eri asetuksiin, jotka voivat vaikuttaa Microsoft 365 -työkuormien kuten Teamsin, Office 365 -ryhmien, SharePointin ja OneDriven jakamiseen organisaatiosi ulkopuolisten käyttäjien kanssa. Nämä asetukset on sijoitettu Azuren Active Directoryn, Microsoft 365:n, Teamsin ja SharePointin hallintakeskuksiin.

- [Microsoft 365 -vieraskäyttäjien asetukset](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-guest-settings?view=o365-worldwide)
