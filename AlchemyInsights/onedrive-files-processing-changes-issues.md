# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: vian määritys "muutosten käsittelyyn"

Jos näet "käsittely muutokset" pitkäksi ajaksi, se voi johtua siitä, että sinulla on online-tiedosto auki, paljon tiedostoja juuttunut jonoon tai erittäin suuri tiedostojen synkronointi, katso [OneDrive juuttunut käsittelyyn](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Lue tämä artikkeli, jos haluat tehdä muutoksia tekstin käsittelyyn. Näet tämän sanoman myös, jos sinulla on tiedosto, joka muuttuu koko ajan (kuten Outlook. PST-tiedoston tai tieto kannan). Lue, miten voit siirtää säännöllisesti käyttämääsi [Outlookin Data tiedostoja](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) OneDrivesta.
