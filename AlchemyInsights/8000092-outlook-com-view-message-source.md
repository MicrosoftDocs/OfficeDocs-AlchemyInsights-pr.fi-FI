# <a name="how-to-view-message-headers-or-message-source"></a>Viestin otsikoiden tarkasteleminen tai viestin lähde

Voit tarkastella otsikoita ja muita tietoja sähköpostiviestissä, valitse lukuruudun yläkulmassa kolme pistettä ja valitse **Näytä viestin lähde**.
