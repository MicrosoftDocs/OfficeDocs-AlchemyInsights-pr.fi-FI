# <a name="data-loss-prevention-dlp-overview"></a>Tietojen menetyksen estämisen (DLP) yleiskatsaus

DLP-käytännöllä voit tehdä seuraavat:

- Tunnista luottamukselliset tiedot useissa sijainneissa, kuten Exchange Onlinessa, SharePoint Onlinessa ja OneDrive for Businessissa.


Voit esimerkiksi tunnistaa minkä tahansa tiedoston, joka sisältää mihin tahansa OneDrive for Businessin sivustoon tallennetun luotto kortin numeron, tai voit valvoa vain tiettyjen henkilöiden OneDrive-sivustoja.

- Estä luottamuksellisten tietojen tahaton jakaminen.


Lisä tietoja on seuraavassa [artikkelissa](https://docs.microsoft.com/microsoft-365/compliance/data-loss-prevention-policies).
