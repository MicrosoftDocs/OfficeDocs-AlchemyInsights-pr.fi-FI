# <a name="transfer-ownership-of-a-microsoft-form"></a>Microsoft-Form omistusoikeuden siirtäminen

Siirrät tutkimuksen, tentin tai kyselyn Microsoft Forms-ryhmään niin kaikkien ryhmän jäsenten tulee muodossa omistajille. Voit myös siirtää lomakkeen omistajuuden toiselle käyttäjälle Jos Edellinen omistaja on lähtenyt organisaatiosta.

Lisätietoja [lomakkeen omistusoikeuden siirtäminen](https://support.office.com/article/Transfer-ownership-of-a-form-921a6361-a4e5-44ea-bce9-c4ed63aa54b4).
