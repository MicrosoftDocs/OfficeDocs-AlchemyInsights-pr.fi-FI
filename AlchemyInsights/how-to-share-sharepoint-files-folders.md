# <a name="how-to-share-sharepoint-files-or-folders"></a>SharePoint-tiedostojen tai-kansioiden jakaminen

SharePoint-sivustoon tallentavat tiedostot ovat yleensä kaikkien niiden käytettävissä, joilla on sivuston käyttö oikeudet, mutta haluat ehkä jakaa tiettyjä tiedostoja tai kansioita sellaisten henkilöiden kanssa, joilla ei muuten ole sivuston käyttö oikeutta. Kun jaat tiedostoja ja kansioita, voit päättää, antaako se ihmisten muokata vai vain tarkastella niitä.
