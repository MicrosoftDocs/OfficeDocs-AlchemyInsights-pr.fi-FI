# <a name="generate-automatic-captions-and-a-transcript-in-microsoft-stream"></a>Luo automaattiset kuva tekstit ja transkriptio Microsoft streamissä

Streamista voi automaattisesti luoda kuva tekstejä automaattisella puheen tunnistus tekniikalla, ja voit tarkastella vieritettävä teksti-ikkunaa, jossa voit helposti siirtyä ja seurata videon puhutun sisällön mukana.

- [Luo automaattiset kuva tekstit ja transkriptio Microsoft Streamin videoille](https://docs.microsoft.com/stream/portal-autogenerate-captions)

- [Videon vieritys teksti ikkunan näyttäminen Microsoft streamissä](https://docs.microsoft.com/stream/portal-configure-transcript-mode)
