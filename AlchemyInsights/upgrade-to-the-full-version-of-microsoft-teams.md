# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Päivitä Microsoft Teamsin täysversioon

Haluatko päivittää maksuttomasta Microsoft Teamsin täysversioon? Katso [Microsoft Teamsin maksuttoman microsoft 365 -tilausversion päivittäminen](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Sinun on täytettävä seuraavat vaatimukset:

- Olet henkilö, joka on rekisteröitynyt Teamsin nykyiseen ilmaiseen versioon.
- Jos sinulla on oma toimialue, sitä ei voi vielä liittää Azure Active Directoryyn.
- Jos haluat siirtää tietoja päivityksen yhteydessä, sinun on tilattava päivityksen aikana. Jos ostit tilauksen päivitysprosessin ulkopuolella, et voi siirtää tietoja.
