# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Päivitä Microsoft teamsin täysversioon

Haluatko päivittää Microsoft teamsin ilmaisesta täydellisestä versiosta? Katso [Microsoft teamsin päivittäminen microsoft 365-tilaus versioon](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Sinun on täytettävä seuraavat vaatimukset:

- Olet henkilö, joka on rekisteröitynyt teamsin nykyiseen ilmaiseen versioon.
- Jos sinulla on oma toimi alue, sitä ei voi jo yhdistää Azure Active Directoryyn.
- Jos haluat siirtää tietoja päivitettäessä, sinun on tilattava päivitys prosessin aikana. Jos ostit tila uksen päivitys prosessin ulkopuolelta, et voi siirtää tietoja.
