# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>Koontinäyttö ei vastaa muutoksia päivityksen jälkeen

Jos koontinäyttö ei vastaa muutoksia päivityksen jälkeen, odota noin 10–15 minuuttia, jotta päivitys näkyy koontinäytön ruuduissa. Jos visualisointi ei vieläkään näy, kiinnitä visualisointi uudelleen koontinäyttöön.

Lisätietoja on kohdassa [Päivitysskenaarioiden vianmääritys](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios).
