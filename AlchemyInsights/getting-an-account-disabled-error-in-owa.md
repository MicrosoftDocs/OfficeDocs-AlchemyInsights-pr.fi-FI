# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Poistetaanko tili käytöstä Outlookin verkkoversiossa?

Jos saat virheilmoituksen Tilisi on `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` poistettu **käytöstä,** kun yrität avata Outlookin verkkokäyttöä (aiemmin Outlook Web App), organisaation järjestelmänvalvoja on saattanut poistaa Outlookin verkkokäytön käytöstä. Järjestelmänvalvoja voi korjata tämän virheen noudattamalla ohjeaiheen [Ota Outlook Web App in postilaatikkokäyttöön tai poista se käytöstä](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)ohjeita.
