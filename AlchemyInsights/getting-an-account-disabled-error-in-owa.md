# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Saatko tilin, joka on poistettu käytöstä Outlookin verkko versiossa?

Jos saat virhe ilmoituksen, jonka mukaan **tilisi on** poistettu käytöstä  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` , kun yrität avata Outlookin verkko version (aiemmalta nimeltään Outlook Web App), organisaatiosi järjestelmänvalvoja on saattanut poistaa käytöstä Outlookin verkko version käytön. Järjestelmänvalvoja voi korjata tämän virheen noudattamalla ohje aiheen [Outlook Web Appin ottaminen käyttöön tai poistaminen käytöstä](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
