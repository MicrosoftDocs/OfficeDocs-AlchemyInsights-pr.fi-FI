# <a name="sharepoint-online-calendar-overlay"></a>SharePoint Online-kalenterin peitto kuva

Voit lisätä toisen SharePoint-kalenterin päällekkäin niin kauan kuin molemmat SharePoint-kalenterit ovat samassa sivustokokoelmassa. Exchange Onlinesta noudetun SharePoint Onlinen tai SharePoint Serverin kalenterin peitto kuvaa ei voi käyttää.

SharePoint Online tarjoaa modernin käyttö kokemuksen Microsoft 365-ryhmä kalentereissa. Lisä tietoja on artikkelissa [Ryhmä kalenteri-verkko-osan käyttäminen](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).
