# <a name="using-office-product-keys"></a>Office-tuote avainten käyttäminen

**Office-tuote avaimet**

Käytä Microsoft-tiliäsi tuote avaimesi sijaan, jotta voit asentaa ja aktivoida Officen ja ostaa erikseen Microsoft 365-sovelluksia, kuten Projectin, Vision, Wordin, Excelin ja Outlookin. Lisä tietoja on Ohje aiheessa [Officen kehotukset tuote avaimen](https://support.office.com/article/12a5763a-d45c-4685-8c95-a44500213759?ui=en-US&rs=en-US&ad=US#bkmk_promptforpkey) käyttöön

- Jos Office-ostokseesi tuli tuote avain, Katso lisä tietoja artikkelista [Office-tuote avaimen lisääminen](https://support.office.com/article/Where-to-enter-your-Office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db).

- Ohjeita Office 365 Home-tai henkilökohtaiseen tuote avaimeen on kohdassa [tuote avainten käyttäminen Officessa](https://support.office.com/article/using-product-keys-with-office-12a5763a-d45c-4685-8c95-a44500213759).

**Office-tila uksen Akti voiminen** 

- Jos ostit Microsoft 365-tila uksen jälleenmyyjältä, Katso lisä tietoja artikkelista [microsoft 365-sovellusten yritys version tuote avaimen lisääminen](https://docs.microsoft.com/microsoft-365/commerce/enter-your-product-key).

- Jos näyttöön tulee virhe sanoma, kun yrität lunastaa tuote avaimen, Katso lisä tietoja artikkelista [ongelmia Microsoft 365 for Businessin tuote avaimen kanssa?](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions)
