# <a name="block-pop-up-windows-in-microsoft-edge"></a>Ponnahdusikkunoiden estäminen Microsoft Edgessä

1. Siirry... **(Asetukset ja muuta)**  >  **Asetukset**  >  **Sivuston käyttöoikeudet.**
2. Valitse **Ponnahdusikkunat ja uudelleenohjaukset.**
3. Ota **estäminen käyttöön.**
