# <a name="windows-fresh-start"></a>Windowsin tuore alku

Windowsin Fresh Start -laitteen toiminto tarjoaa vaihtoehdon Laitteen pyyhintä- ja laite pois käytöstä -toiminnoille sekä mahdollisuuden säilyttää laitteen rekisteröintitila käyttäjän kotikansion sovellusten ja asetusten kanssa.

Lisätietoja on ohjeaiheessa [Windows 10 -laitteiden palauttaminen Intune -toiminnolla.](https://docs.microsoft.com/intune/device-fresh-start)
