# <a name="how-to-change-your-theme"></a>Teeman muuttaminen

Voit muuttaa teeman Outlook.com-tai Outlook Web- **asetukset** ja valitse teema **Teema**-kohdasta. Jos haluat nähdä kaikki käytettävissä olevat teemat, valitse **Näytä kaikki** **teeman**mukaisesti.
