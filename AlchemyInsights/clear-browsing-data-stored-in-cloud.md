# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>Pilvipalveluun tallennettujen selaustietojen tyhjentäminen (jos olet ottanut synkronoinnin tai mukauttaminen käyttöön)

Voit tarkastella, hallita tai poistaa tietoja, jotka olet tallentanut pilvipalveluun, käyttämällä tietosuojan koontinäyttöä. Koontinäytössä poistamistasi tiedoista ei poisteta laitteestasi. Lisätietoja on kohdassa [Tietojen tarkasteleminen tietosuojan koontinäytössä.](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433)
