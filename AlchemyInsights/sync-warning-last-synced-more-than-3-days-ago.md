# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>Synkronointivaroitus: viimeksi synkronoitu yli 3 päivää sitten

Lisätietoja vianmäärityksestä on [ohjeaiheessa Hakemistosynkronointiin liittyvien ongelmien korjaaminen](https://docs.microsoft.com/office365/enterprise/fix-problems-with-directory-synchronization).

Voit tarkistaa synkronoinnin tilan Office-hallintaportaalissa noudattamalla kohdassa [Hakemistosynkronoinnin tilan tarkasteleminen annettuja](https://docs.microsoft.com/office365/enterprise/view-directory-synchronization-status)ohjeita.
