# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>Synkronoinnin Varoitus: synkronoitu viimeksi yli 3 päivää sitten

Lisä tietoja vian määrityksestä on kohdassa [Hakemisto synkronointiin liittyvien ongelmien korjaaminen](https://docs.microsoft.com/office365/enterprise/fix-problems-with-directory-synchronization).

Voit tarkistaa synkronoinnin tilan Office-hallinta portaalissa noudattamalla [hakemiston synkronoinnin tila](https://docs.microsoft.com/office365/enterprise/view-directory-synchronization-status)-kohdan ohjeita.
