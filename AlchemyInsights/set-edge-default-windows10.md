# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Microsoft Edgen asetaminen oletusselaimeksi Windows 10 -laitteessa

Toimi seuraavasti:

1. Valitse **aloituspainike** ja kirjoita sitten **Oletussovellukset.**

1. Valitse hakutuloksista **Oletussovellukset.**

1. Valitse **verkkoselaimessa** **Microsoft Edge.**
