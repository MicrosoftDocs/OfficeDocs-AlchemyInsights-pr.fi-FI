# <a name="create-a-shared-mailbox"></a>Jaetun postilaatikon luominen

Jaetun posti laatikon avulla useampi kuin yksi henkilö voi lähettää ja vastaanottaa sähkö postia aliakselle, kuten sales@contoso.com, määrittelemättä toista käyttö oikeutta. 

Katso artikkeli jaetun [Posti laatikon luominen](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) vinkkejä ja lisä tietoja jaetun posti laatikon käyttämisestä.
