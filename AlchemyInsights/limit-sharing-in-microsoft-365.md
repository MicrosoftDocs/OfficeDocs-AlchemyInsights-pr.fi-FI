# <a name="limit-sharing-in-microsoft-365"></a>Jakamisen rajoittaminen Microsoft 365-sovelluksessa

Vaikka et voi poistaa sisäistä jakamista kokonaan käytöstä tai poistaa Jaa-painiketta sivustosta, voit rajoittaa jakamista Microsoft 365-sovelluksessa organisaatiosi tarpeiden mukaan. Lisä tietoja on kohdassa [jakamisen rajoittaminen Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)-sovelluksessa.
