# <a name="change-the-font-size-in-a-message"></a>Viestin fontti koon muuttaminen

1. Jos olet jo kirjoittanut viestin, valitse teksti.
2. Valitse viestin alla oleva  **muoto** -kuvake (se on ensimmäinen kuvake).
3. Valitse viestin yläpuolella oleva  **fontti koko**  -kuvake (se on seitsemäs kuvake).
4. Valitse koko.
