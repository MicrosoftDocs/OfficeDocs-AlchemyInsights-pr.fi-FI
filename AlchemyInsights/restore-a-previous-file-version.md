# <a name="restore-a-previous-file-version"></a>Aiemman tiedostoversion palauttaminen

Online-versiohistorian avulla voit tarkastella ja palauttaa tiedostojen vanhempia versioita OneDrivessa tai SharePointissa. Versiohistoria toimii kaikkien tiedostotyyppien kanssa, mukaan lukien PDF-tiedostot, CAD-tiedostot, valokuvat ja videot. Edellisen tiedostoversion palauttaminen:
  
1. Avaa SharePoint-tiedostokirjasto tai OneDrive-sijainti.
    
2. Valitse tiedosto napsauttamalla sitä ja valitse sitten **Versiohistoria**. Jos komentopalkissa ei ole versiohistoriaa, valitse Lisää ( **...** ). 
    
3. Valitse versio, jota haluat tarkastella, tai valitse version oikealla puolella Lisää ( **...** ) ja valitse sitten **Palauta**.
