# <a name="restore-a-previous-file-version"></a>Aiemman tiedosto version palauttaminen

Online-versio historian avulla voit tarkastella ja palauttaa tiedostojen vanhempia versioita OneDrivessa tai SharePointissa. Versio historia toimii kaikkien tiedosto tyyppien, kuten PDF:ien, CAD-tiedostojen, valo kuvien ja videoiden kanssa. Aiemman tiedosto version palauttaminen:
  
1. Avaa SharePoint-asiakirja kirjasto tai OneDrive-sijainti.
    
2. Valitse tiedoston vasemmalla puolella oleva tiedosto ja valitse sitten **versio historia**. Jos et näe versio historiaa komento palkissa, valitse Lisää ( **...** ). 
    
3. Valitse versio, jota haluat tarkastella, tai napsauta version oikealla painikkeella Lisää ( **...** ) ja valitse sitten **Palauta**.
