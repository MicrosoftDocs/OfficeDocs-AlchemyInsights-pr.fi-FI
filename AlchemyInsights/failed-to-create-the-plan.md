# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>Suunnitelman luominen epäonnistui -virhe yritettäessä luoda Planner-välilehteä Microsoft Teamsissa

Jos saat Suunnitelman luominen epäonnistui -virheilmoituksen, kun yrität luoda Planner-välilehteä Microsoft Teamsissa, varmista, että järjestelmänvalvoja on ryhmän jäsen seuraavasti:

1. Valitse hallintakeskuksessa   >  [Ryhmät.](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups) 
2. Valitse ryhmä ja valitse sitten **Jäsenet-välilehdessä** **Näytä kaikki ja hallitse jäseniä**.
3. Lisää jäsen ja valitse **Tallenna**.
