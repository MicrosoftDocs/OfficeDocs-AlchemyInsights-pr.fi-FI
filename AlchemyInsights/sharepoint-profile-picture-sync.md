# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Milloin profiilini muutokset synkronoidaan SharePoint-käyttäjäprofiilisovellukseen?

SharePoint Onlinen profiilikuvien synkronointi edellyttää käyttäjän toimia, jotta hänen kuvansa synkronoidaan. Valokuvia ei synkronoida sharePoint Onlineen automaattisesti. SharePoint Online synkronoi vain Exchange Onlinen valokuvia, ei Azure Active Directorya.

Jos käyttäjä ei ole koskaan suorittanut toimintoa, joka kehottaa synkronointia tai hän on päivittänyt profiilikuvansa eikä uusi toiminto näy SharePoint Onlinessa, käyttäjän on ehkä kehotettava sitä synkronoimaan. Näin he voivat tehdä tämän:

1. Avaa selain In-Private- tai Incognito-istunnossa ja [siirry kohtaan Office.com](https://www.office.com/).

2. Kirjaudu sisään työpaikan tai oppilaitoksen tilillä ja valitse OneDrive- tai SharePoint-ruutu.

3. [Lisää heidän profiilikuvansa Microsoft 365 :ään.](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7)

[Tietoja profiilikuvien synkronoinnista Microsoft 365:ssä](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
