# <a name="cant-create-a-team-in-teams"></a>Tiimiä ei voi luoda tiimeissä

Teamsin järjestelmänvalvoja ei ehkä ole antanut sinulle tarvittavia käyttö oikeuksia teamsin luomiseen.  

Lisä tietoja on kohdassa organisaation käyttö oikeus ryhmän asetusten luominen tai muuttaminen Microsoft 365-sovelluksessa. Jos olet lisännyt käyttö oikeus ryhmään, voit luoda tiimejä.
