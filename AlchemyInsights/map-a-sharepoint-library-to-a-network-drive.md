# <a name="map-a-sharepoint-library-to-a-network-drive"></a>SharePoint-kirjaston Yhdistä verkko asemaan

Kirjaston yhdistäminen verkko asemiksi on väliaikaista, ja sitä tuetaan vain Internet Explorerin avulla. Synkronoi sen sijaan [SharePoint-tiedostot uuden OneDrive-synkronointi ohjelman kanssa](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx), joka tarjoaa [tiedostot](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx)pyydettäessä. Käyttää kaikkia tiedostojasi OneDrivessa ilman paikallista tallennus tilaa.
  
[Ohjeita yhdistettyjen verkko asemien määrittämiseen ja vian määritykseen](https://go.microsoft.com/fwlink/?linkid=872946)
