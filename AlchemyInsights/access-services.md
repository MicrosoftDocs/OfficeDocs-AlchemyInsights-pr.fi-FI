# <a name="access-services"></a>Access Services

SharePoint Online-Access Services-ominaisuus on poistettu. Uuden Access-pohjainen web-sovellukset ja WWW-tietokantoja ei voi luoda ja olemassa olevan web-sovellukset ja WWW-tietokantoja suljetaan huhtikuussa, 2018 mukaan. Tämä muutos ei koske Access työpöydän tietokannan (.accdb). Microsoft suosittelee Microsoft Access vaihtoehtona PowerApps web-sovellukset ja WWW-tietokantoja. 
  
[Lisätietoja Microsoftin PowerApps](https://powerapps.microsoft.com/)
