# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Lataa Microsoft teams-koko uksen tallennus Streamille (a1)

Useimmille Microsoft 365-käyttäjille teamsin Kokous tallenteet ladataan automaattisesti Microsoft Streamille koko uksen jälkeen. Kuitenkin  **elokuun 20, 2020**, kun käyttäjät, joilla on a1 (mukaan lukien a1 Plus)-käyttö oikeudet, aloittavat teamsin Kokous tallennuksen, tämä toiminto muuttuu.  

Lisä tietoja on Ohje aiheessa [Microsoft teamsin Kokous tallennuksen lataaminen Streamille](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording).
