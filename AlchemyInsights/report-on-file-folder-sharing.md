# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Raportin tiedostojen ja kansioiden jakamisesta SharePoint-sivustossa

Voit luoda CSV-tiedoston, joka määrittää jokaisen yksilöllisen tiedoston, käyttäjän, käyttö oikeuden ja linkin tiettyyn SharePoint-sivustoon tai OneDrive-kirjastoon. Tämä voi auttaa ymmärtämään, miten jakaminen on käytössä ja ovatko tiedostot tai kansiot jaettu vieraille.

Jos haluat lisä tietoja, [raportoi tiedostojen ja kansioiden jakamisesta SharePoint-sivustossa](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Tämä toiminto on tällä hetkellä käytössä Ensijulkaisuvuokralaisille, ja se siirtyy vähitellen kaikille tuotannon vuokraajille.
