# <a name="how-to-select-all-in-outlookcom"></a>Valitse kaikki-vaihto Outlook.com

Voit valita vain kaikki, jos kaikki viestit ovat näkyvissä. Tämä estää vahingossa tapahtuvia toimintoja viesteistä, joita et näe.

**CTRL + napsautus** valitsee useita yksittäisiä viestejä.

**Vaihto + napsautus** valitsee kaikki viestit ensimmäisen ja viimeisen valitsemasi viestin väliltä.
