# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Laajennuksen lisääminen Microsoft Edgeen Chrome Web Storesta

Jos haluat lisätä laajennuksen Microsoft Edgeen Chrome Web Storesta, toimi seuraavasti:

1. Siirry Chrome Web Storeen.
2. Valitse sivun yläreunassa olevassa nauhassa **Salli laajennukset muista kaupoista**.
3. Valitse laajennus, jonka haluat lisätä, ja valitse sitten **Lisää Chromeen**.
4. Tarkista vahvistus valinta ikkunassa huolellisesti laajennuksen mahdollisesti tekemät muutokset ja valitse sitten **Lisää tunniste**.
Viimeisessä valinta ikkunassa vahvistetaan, että laajennus on lisätty.
