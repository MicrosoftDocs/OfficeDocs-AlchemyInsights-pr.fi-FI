# <a name="set-passwords-to-never-expire"></a>Aseta salasanat vanhenemaan 

Jos haluat asettaa KAIKKI salasanat vanhenemaan, siirry Microsoft 365 -hallintakeskuksessa **Settings** > [Security &amp; Privacy -sivulle.](https://portal.office.com/adminportal/home#/settings/security) Valitse **Salasanakäytäntö-kohdan**vierestä **Muokkaa** ja valitse Käytössä **-liukusäädin.**
  
Lisätietoja on kohdassa: [Salasanan vanhentumiskäytännön määrittäminen organisaatiollesi.](https://docs.microsoft.com/office365/admin/manage/set-password-expiration-policy)
  
[Aseta salasana, joka ei vanhene koskaan tietyille käyttäjille](https://docs.microsoft.com/office365/admin/add-users/set-password-to-never-expire)
