# <a name="how-to-change-your-theme"></a>Teeman muuttaminen

Vaihtaa teeman Outlook.com-tai Outlook web-sivulla, siirry **asetukset**![asetukset](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) ja valitse teema **Teema**-kohdasta. Jos haluat nähdä kaikki käytettävissä olevat teemat, valitse **Näytä kaikki** **teeman**mukaisesti.
