# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable määritettäessä tunnistetiedot

GatewayNotReachable saattaa ilmetä yritettäessä määrittää tietolähteen tunnistetietoja. Tämä saattaa aiheutua vanhentunutta yhdyskäytäväpalvelimen. Asenna uusin yhdyskäytävä ja yritä uudelleen.

Lisätietoja:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
