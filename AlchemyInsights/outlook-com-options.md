# <a name="how-to-find-options-in-outlookcom-settings"></a>Miten asetukset löytyy Outlook.com-asetukset

Valitse sivun yläosassa pyydykset kuvaketta asetukset. Valitse Asetukset-ruudun alaosasta **Näytä kaikki Outlookin asetukset**.

Tämä toiminto näyttää, General, sähköposti, kalenteri, ja (tarvittaessa) ja Premium asetukset.
