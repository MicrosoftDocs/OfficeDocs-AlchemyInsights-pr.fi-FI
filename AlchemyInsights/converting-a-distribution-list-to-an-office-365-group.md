# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Jakeluryhmän muuntaminen Microsoft 365 -ryhmäksi

Voit muuntaa jakeluryhmät Microsoft 365 -ryhmiin, jos ne täyttävät tietyt ehdot. Lisätietoja ryhmän muuntamisesta tai muunto-ongelmien vianmäärityksestä on [ohjeaiheessa Jakeluluetteloiden päivittäminen Microsoft 365 -ryhmiin Outlookissa](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).
