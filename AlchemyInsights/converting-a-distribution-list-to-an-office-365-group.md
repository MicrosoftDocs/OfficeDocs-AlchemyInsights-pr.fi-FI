# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Distribution Groupin muuntaminen Microsoft 365-ryhmäksi

Voit muuntaa jakelun ryhmät Microsoft 365-ryhmiksi, jos ne täyttävät tietyt ehdot. Jos haluat muuntaa ryhmän tai suorittaa muunto ongelmia, Katso lisä tietoja artikkelista [jakelun luettelojen päivittäminen Microsoft 365-ryhmiin Outlookissa](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).
