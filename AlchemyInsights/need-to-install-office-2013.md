# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Office 2013 ei enää tueta Microsoft 365 -tilauksissa

28.2.2017 lähtien Microsoft 365 -tilauksia ei enää tueta Office 2013:ssa.
  
Voit varmistaa Office-tuotepaketin turvallisuuden ja saada kaikki uusimmat ominaisuudet asentamalla Officen uusimman version ilmaiseksi osana Microsoft 365 -tilaustasi. Lisätietoja [on ohjeaiheessa Käyttäjien päivittäminen uusimpaan Office-asiakasohjelmaan](https://docs.microsoft.com/microsoft-365/admin/setup/upgrade-users-to-latest-office-client).
