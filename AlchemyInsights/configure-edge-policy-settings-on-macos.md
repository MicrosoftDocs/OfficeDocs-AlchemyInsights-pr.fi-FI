# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Microsoft Edgen käytäntöasetusten määrittäminen MacOS:ssä Jamfin avulla

Jos haluat määrittää Microsoft Edgen (vakaa kanava 81 tai uudempi) macOS:ssä, käytä Microsoft Edgen käytännön luettelotiedostoa Jamf Pro 10.19:ssä.

Lisätietoja on kohdassa [Microsoft Edgen käytäntöasetusten määrittäminen MacOS:ssä Jamfin avulla.](https://go.microsoft.com/fwlink/?linkid=2134761)
