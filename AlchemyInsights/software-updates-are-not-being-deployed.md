# <a name="software-updates-are-not-being-deployed"></a>Ohjelmisto päivityksiä ei ole otettu käyttöön

Jos olet määrittänyt ohjelmisto päivitykset Windows 10-päivitys renkaalla, mutta päivityksiä ei ole otettu käyttöön, kokeile seuraavaa:  

- Harkitse Windows-ylläpidon vaihtamista  **puolivuosittaisista kanava**  versiosta tiukempaan, useammin julkaistavan julkaisun tyyppiin.
- Tarkista  **laatu päivityksen**  ja  **ominaisuuksien päivityksen**lykkäys vaihe. Lykkäys aika voi aiheuttaa päivitysten viivästymisen enintään 180 päiväksi.
