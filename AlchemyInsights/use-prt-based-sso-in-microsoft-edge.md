# <a name="use-prt-based-sso-in-microsoft-edge"></a>PRT-pohjaisen SSO:n käyttö Microsoft Edgessä

Ensisijainen [päivitystunnus (PRT)](https://go.microsoft.com/fwlink/?linkid=2133632) on Azure Active Directory -avain, jota käytetään todentamiseen Windows 10-, iOS- ja Android-laitteissa. PRT ottaa SSO:n käyttöön kaikissa näissä laitteissa käytetyille sovelluksille.

Microsoft Edgessä PRT-pohjaisen SSO:n tuki on alkuperäisenä eikä edellytä lisälaajennuksia. Windows 10 RS3:ssa ja sitä uudempien käyttäjissä kirjautuneet käyttäjät saavat PRT:n kautta sso-sso-käyttöjärjestelmän sivustoille, jotka tukevat PRT-pohjaista SSO:ta.
