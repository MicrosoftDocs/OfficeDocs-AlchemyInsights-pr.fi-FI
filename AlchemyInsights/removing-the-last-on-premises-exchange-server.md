# <a name="removing-the-last-on-premises-exchange-server"></a>Viimeisimmän paikallisen Exchange-palvelimen poistaminen

Ennen kuin poistat viimeisimmän Exchange-palvelimen paikallisesta organisaatiosta, sinun tulee varmistaa, että ymmärrät tämän tekemisen seura ukset, jotta voit päättää, tarvitaanko tätä viimeistä palvelinta muihin tarkoituksiin. Lisä tietoja on Ohje [aiheessa paikallisten Exchange-palvelinten käytöstäpoistaminen yhdistelmä ympäristössä](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
