# <a name="removing-the-last-on-premises-exchange-server"></a>Poistetaan viimeinen paikalliseen Exchange-palvelimeen

Varmista ennen poistamista viimeisen Exchange-palvelimeen organisaation tiloissa, ymmärrät vaikutukset näin auttaa päättämään, jos tarvitset viime palvelimen muihin tarkoituksiin. Saat lisätietoja, tutustu [miten ja milloin poistaminen käytöstä paikalliseen Exchange-palvelinten käyttöönoton hybridi](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
