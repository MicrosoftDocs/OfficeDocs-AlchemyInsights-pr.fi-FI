# <a name="copy-and-paste-in-outlookcom"></a>Kopioi ja liitä Outlook.com-

Outlook.com ei kopioi tai Liitä-painiketta, mutta voit käyttää pikanäppäimiä:

**PC:** CTRL + C kopioi, CTRL + X Leikkaa ja liitä CTRL + V.

**-Mac:** CMD + C kopioidaksesi CMD + X Leikkaa ja liitä CMD + V.
