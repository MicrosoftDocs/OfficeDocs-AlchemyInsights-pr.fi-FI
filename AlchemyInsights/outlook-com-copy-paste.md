# <a name="copy-and-paste-in-outlookcom"></a>Kopioiminen ja liittäminen Outlook.com

Outlook.com ei ole kopiointi- tai liittämispainikkeita, mutta voit käyttää pikanäppäimiä:

**Tietokoneessa:** CTRL+C kopioida, CTRL+X leikata, ja CTRL + V liittää.

**Macissa:** CMD + C kopioida, CMD + X leikata, ja CMD + V liittää.
