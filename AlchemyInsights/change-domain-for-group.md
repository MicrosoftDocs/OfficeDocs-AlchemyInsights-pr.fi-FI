# <a name="change-the-domain-for-microsoft-365-group"></a>Microsoft 365 Groupin toimialueen vaihtaminen

Jos haluat muuttaa Microsoft 365 -ryhmän toimialuetta luomisen aikana, noudata microsoft [365 -ryhmien luomisessa käytettävien toimialueiden valitseminen](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups)-kohdan ohjeita.
  
Kun Microsoft 365 -ryhmä on jo luotu, toimialuetta ei tällä hetkellä voi muuttaa. Ainoa tapa on luoda uusi Microsoft 365 -ryhmä ja poistaa vanha ryhmä tarvittaessa (lisää sähköpostiosoitteita voidaan lisätä [Exchange-hallintakeskuksen](https://outlook.office365.com/ecp.aspx)kautta ).
