# <a name="sharepoint-publishing-infrastructure"></a>SharePoint-julkaisemisen infrastruktuuri


<p>SharePoint sisältää ominaisuuksia, joiden avulla voit hallita ja julkaista intranet-WWW-sivujen rich classic. Aktivoi SharePoint Publishing infrastruktuurin sivustokokoelman tasolla ottaa ne käyttöön.</p>  <p>Saat lisätietoja käymällä alla olevia linkkejä.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Julkaisuominaisuuksien käyttöönottaminen</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Julkaisuominaisuudet</a></li>  </ul>  <p>Huomautus:</p>  <ul>  <li>Perinteinen publishing sivustomallit sisältävät oletusarvon mukaan julkaistaan ominaisuudet.</li>  <li>Perinteinen julkaisuominaisuudet eivät ole käytettävissä nykyisissä sivustoissa. Moderni sivustot tarjoavat uudempi vaihtoehto oletusarvon mukaan.</li>  </ul>
