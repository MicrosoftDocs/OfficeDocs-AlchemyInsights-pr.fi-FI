# <a name="add-or-remove-a-custom-role"></a>Mukautetun roolin lisääminen tai poistaminen

Lisä tietoja Azure-resurssien Hallin nasta on Ohje aiheessa [Azure-rooli määritysten lisääminen tai poistaminen Azure-portaalin avulla](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Jos haluat hallita mukautettuja rooleja Azure Active Directoryssa (Azure AD), tutustu Ohje aiheeseen [Mukautetut järjestelmänvalvojien roolit Azure Active Directoryssa (esikatselu)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
