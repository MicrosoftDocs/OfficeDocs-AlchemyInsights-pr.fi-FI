# <a name="create-custom-content-types"></a>Mukautettujen sisältötyyppien luominen

1. Valitse luettelo, jota haluat käsitellä pikakäynnistyksessä, tai valitse **Asetukset** (rataskuvake) ja valitse sitten **Sivuston asetukset**.
2. Valitse **Web Designer Galleries**-kohdasta sivuston **sisältötyypit.**
3. Valitse Sivuston sisältötyypit -sivulla Luo.
4. Täytä Uusi sivuston sisältötyyppi -sivulla mukautetun sisältötyypin tiedot.

Lisätietoja on ohjeaiheessa [Mukautetut sisältötyypit](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
