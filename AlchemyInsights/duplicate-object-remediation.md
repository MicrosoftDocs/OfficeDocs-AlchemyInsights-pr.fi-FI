# <a name="duplicate-object-synchronization-errors"></a>Synkronointivirheitä objekti

Jos näyttöön tulee jokin seuraavista virheistä synkronoinnin aikana:

- Microsoft Online Services-kansiossa on jo synkronoitu objekti saman välityspalvelimen osoite.

- UserPrincipalName samanlainen sanoma.

Uusi AADConnect terveys-Diagnostiikan avulla voit ratkaista samanlaisia virheitä. Lisätietoja on ohjeaiheessa [diagnosointi ja riskien monistettu määritteen synkronointivirheitä parantaminen](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
