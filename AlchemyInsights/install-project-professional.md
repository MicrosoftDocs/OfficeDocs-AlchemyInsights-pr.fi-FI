# <a name="install-project-professional"></a>Project Professionalin asentaminen

Lisätietoja Project Onlinen tilausversion ja Project 2019:n, 2016:n tai 2013:n muun kuin tilausversion asentamisesta on kohdassa [Projectin](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) asentaminen. 

Jos project 2010 tai 2007 on, noudata [Office 2010:n](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) tai [Office 2007:n](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)ohjeita. 

Project myydään erillisenä sovelluksena, joten jos sinulla ei ole sitä vielä, [Osta tai kokeile Projectia](https://products.office.com/project).
