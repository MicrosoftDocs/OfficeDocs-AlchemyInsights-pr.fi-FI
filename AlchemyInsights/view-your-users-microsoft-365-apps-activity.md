# <a name="view-your-users-microsoft-365-apps-activity"></a>Käyttäjien Microsoft 365-sovellusten toiminnan tarkasteleminen

[Officen aktivointi raportin](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) avulla voit tarkastella, ketkä käyttäjät ovat asentaneet ja aktivoineet Microsoft 365-sovelluksia vähintään yhdessä laitteessa.

Lisäksi voit saada näkymän käyttäjien Microsoft 365-sovellusten toimintaan [microsoft 365-sovellusten käyttö-raportin](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)avulla.
