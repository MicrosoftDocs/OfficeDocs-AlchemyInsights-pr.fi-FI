# <a name="fix-windows-update-errors"></a>Windows Update -virheiden korjaaminen

Jos saat virhekoodin Ladattaessa ja asennettaessa Windows Updatea, **päivityksen vianmääritys** voi auttaa ratkaisemaan ongelman.

- [Lataa Windows 10:n päivityksen vianmääritys](https://support.microsoft.com/help/4027322/windows-update-troubleshooter)
- Suorita vianmääritys. Kun se on valmis, on parasta käynnistää laite uudelleen. Tarkista sitten uudet päivitykset valitsemalla [Käynnistä > Asetukset > Päivitys & Tietoturva> Windows Update](ms-settings:windowsupdate).

Jos ne eivät vieläkään toimi, tutustu [Windows Update: USEIN KYSYTTYIHIN KYSYMYKSIIN.](https://support.microsoft.com/help/12373/windows-update-faq)
