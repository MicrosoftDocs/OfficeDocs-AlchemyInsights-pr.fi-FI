# <a name="sharing-options-appear-dim-or-are-not-available"></a>Jakamis asetukset näkyvät himmeinä tai eivät ole käytettävissä

Jos jotkin jakamis asetukset eivät ole käytettävissä, kun yrität jakaa tietoja SharePoint Onlinesta tai OneDrivesta, tutustu Ohje aiheeseen [jakamis asetukset näkyvät harmaina, kun ne jaetaan SharePoint Onlinesta tai OneDrivesta](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).
