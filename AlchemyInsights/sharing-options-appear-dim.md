# <a name="sharing-options-appear-dim-or-are-not-available"></a>Jakamisasetukset näkyvät himmeinä tai eivät ole käytettävissä

Jos jotkin jakamisasetukset eivät ole käytettävissä, kun yrität jakaa tietoja SharePoint Onlinesta tai OneDrivesta, lisätietoja on ohjeaiheessa [Jakamisasetukset näkyvät harmaana jaettaessa SharePoint Onlinesta tai OneDrivesta](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).
