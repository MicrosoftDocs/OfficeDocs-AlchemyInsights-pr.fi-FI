# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>Uutta allekirjoitusta ei voi lisätä tai luoda Outlookissa

Tämä on tunnettu ongelma. Pyrimme korjaamaan sen ja tila päivitetään [Outlookissa jumittuu napsauttamalla Allekirjoitus-painiketta](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).

Voit kiertää tämän ongelman kokeilemalla esiasennettujen Microsoft Office -työpöytäsovellusten asennuksen [poistamista.](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02) 

Jos tämä ei toimi haavoittuvuuden sisältävässä tietokoneessa, kokeile [rekisteripohjaista kiertotapaa](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).
