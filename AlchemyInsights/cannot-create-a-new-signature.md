# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>Ei voi lisätä tai uuden allekirjoituksen luominen Outlookissa

Tämä on tunnettu ongelma. On tarkoitus korjata, ja tilaksi päivitetään [Outlook jumittuu napsauttamalla allekirjoitus-painiketta](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).

Voit kiertää tämän ongelman ensimmäinen [Kokeile toimia](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02) asennuksen esiasennettu Microsoft Office Desktop Apps. 

Jos tämä ei auta sisältävän, yritä [rekisteripohjaiset kiertotapaa](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).
