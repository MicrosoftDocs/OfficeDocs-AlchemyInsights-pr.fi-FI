# <a name="setting-up-visio"></a>Vision määrittäminen

Näyttää siltä, että tarvitset apua Vision määrittämiseen Microsoft 365-tilaukseesi.
  
Huomautus Mac-käyttäjille: Visio-tuote ei tällä hetkellä ole käytettävissä MacOS-järjestelmässä. Jos sinulla on PC-tieto kone, sinun pitäisi pystyä asentamaan ohjelmisto alla olevien ohjeiden avulla.
  
Voit määrittää asennuksen valmiiksi noudattamalla [tämän artikkelin](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) ohjeita.
