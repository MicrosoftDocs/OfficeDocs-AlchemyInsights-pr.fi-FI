# <a name="solutions-for-unsupported-operating-system"></a>Ratkaisut käyttöjärjestelmälle, jota ei tueta

Kun yrität asentaa Microsoft 365:n, näyttöön saattaa tulla **käyttöjärjestelmän tukematta -virhesanoma,** jos laite ei täytä [järjestelmävaatimuksia.](https://products.office.com/office-system-requirements)
  
Microsoft 365 edellyttää seuraavaa:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 tai Windows Server 2008 R2

    TAI

- Mac OS X 10.10 -käyttöjärjestelmä

Jos sait tämän viestin yrittäessäsi asentaa Officea mobiililaitteeseen tai Chromebookiin, katso [lisätietoja ohjeaiheesta Microsoft Officen asentaminen ja suorittaminen Chromebookissa](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
