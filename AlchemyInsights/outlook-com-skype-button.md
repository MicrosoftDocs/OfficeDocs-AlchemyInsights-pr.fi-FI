# <a name="skype-icon-missing"></a>Skype-kuvake puuttuu

Olemme poistaneet Skype-kuvakkeen tilapäisesti Outlook.com osana suunnittelun päivitystä ja ominaisuuksien päivitystä olemme innoissamme, kun siirrymme ulos. Arvostamme kärsivällisyyttäsi samalla kun valmistamme sinulle uuden ja paremman kokemuksen.

Voit edelleen käyttää Skypeä selaimesi kautta [web.skype.com](https://web.skype.com).
