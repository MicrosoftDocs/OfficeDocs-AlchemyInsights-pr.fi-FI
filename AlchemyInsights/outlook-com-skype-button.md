# <a name="skype-icon-missing"></a>Skype-kuvake puuttuu

Olemme olet tilapäisesti poistaa rakenteen päivittäminen ja ominaisuus päivitys, Olemme iloisia saadessamme on liikkuvan osana Outlook.com Skype-kuvaketta. Arvostamme kärsivällisyyttäsi kun olemme valmistella uusia ja parannettuja kokemus sinulle.

Voit silti käyttää Skype kautta selaimella osoitteeseen [web.skype.com](https://web.skype.com).
