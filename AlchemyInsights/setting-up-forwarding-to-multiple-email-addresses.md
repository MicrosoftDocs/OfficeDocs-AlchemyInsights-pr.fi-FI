# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Useat sähköpostiosoitteet välityksen määrittäminen

Jos tarvitset sähköpostin määrittämiseen välittäminen käyttäjäryhmän, jakeluluettelon luominen, useiden käyttäjien lisääminen ja määrittää sitten luetteloon välittäminen. Ensinnäkin hallintakeskukseen, siirry **ryhmiä** > [ryhmät](https://portal.office.com/adminportal/home#/groups).
  
1. Luo ryhmä.

2. Lisätä useita käyttäjiä.

3. Määrittää postilaatikon siirto kyseiseen ryhmään.
