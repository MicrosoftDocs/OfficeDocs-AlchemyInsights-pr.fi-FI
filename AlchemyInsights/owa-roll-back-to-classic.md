# <a name="the-new-outlook-on-the-web"></a>Uusi Outlook web-sivulla

Ei ole enää mahdollista palata perinteisen Outlook web kokemuksia. Jos haluat antaa palautetta uusi kokemus, siirry [UserVoice foorumi](https://outlook.uservoice.com/forums/313228--outlook-on-the-web-office-365). Voit äänestää ideoita tai ongelmia, jotka haluat nähdä osoitettu ja olemme vastata suoraan top kommentit ja ehdotukset.
