# <a name="configure-and-validate-microsoft-defender-antivirus-network-connections"></a>Microsoft Defenderin virustentorjuntaverkon yhteyksien määrittäminen ja vahvistaminen

Jotta Voit varmistaa, että Microsoft Defenderin virustentorjunta voidaan päivittää, sinun on määritettävä verkko sallimaan päätepisteiden ja tiettyjen Microsoft-palvelinten väliset yhteydet. Lisätietoja on kohdassa [Microsoft Defenderin virustentorjuntaverkon yhteyksien määrittäminen ja vahvistaminen.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/configure-network-connections-microsoft-defender-antivirus)
