# <a name="moving-a-folder-in-outlookcom"></a>Outlook.com-kansion siirtäminen

Voit vetää uuden paikan Outlook.com-kansion Suosikit-hierarkiassa tai voit **siirtää** vaihtoehdon classic Outlook.com pikavalikosta. 
  
Voit järjestää **Suosikit**-kansiot uudelleen, mutta ei voi järjestää uudelleen tai tehdä ylimmän tason kansion alikansio Outlook.com-Beta. Jos teet vahingossa siirtää niin, että se tulee alikansion ylimmän tason kansiota, Luo uusi kansio ja kaikki sähköpostin kopioiminen uuteen ylimmän tason kansioon alikansio. 
  
Lue lisää siitä, miten [siirtää tai nimetä uudelleen kansio Outlook.com](https://support.office.com/article/c9c66fed-8a7c-426a-afc6-0d46a72080fb).
