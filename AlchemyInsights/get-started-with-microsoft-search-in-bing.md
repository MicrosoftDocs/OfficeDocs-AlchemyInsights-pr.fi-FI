# <a name="get-started-with-microsoft-search-in-bing"></a>Microsoft Searchin käytön aloittaminen Bingissä

Jos organisaatiosi on uusi Microsoft Search Bingissä, suosittelemme, että tarkistat [Microsoft Searchin Bing Admin Essentials](https://go.microsoft.com/fwlink/p/?linkid=2127979)-oppaasta, jossa on tietoja Microsoft Searchin määrittämisestä, määrittämisestä ja käyttöönotosta Bingissä.

Jos haluat lisätä käyttäjien tuntemusta Microsoft Searchista [](https://go.microsoft.com/fwlink/p/?LinkID=2114710) Bingissä, viittaa heidät käyttöönottopakettiin (.zip-tiedosto), joka tarjoaa koulutus- ja viestintäresursseja.

Lisätietoja ja työkaluja on ohjeaiheessa [Microsoft Searchin tehosteminen Bing-tuloksissa organisaatiossa.](https://go.microsoft.com/fwlink/?linkid=2152022)
