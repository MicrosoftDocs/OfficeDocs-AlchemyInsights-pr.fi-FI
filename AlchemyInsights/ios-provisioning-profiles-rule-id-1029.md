# <a name="ios-provisioning-profiles"></a>iOS:n valmisteluprofiilit

Apple iOS:n liiketoimintalinjalla käytetään valmisteluprofiilin käsitettä, jota käytetään sovelluksen eheyden vahvistamiseen ja profiilissa määritettyjen käytäntöjen täytäntöönpanoon. Nämä valmisteluprofiilit vanhenevat 12 kuukauden kuluttua, joten käyttöön otettuun sovellukseen on liitettävä uusi profiili, jotta sovellukset voivat jatkaa toimintaasi.
  
Seuraavassa artikkelissa kuvataan, miten voit luoda uuden valmisteluprofiilin intunen kautta käyttöön otettuun iOS LOB -sovellukseen: [iOS-sovellusten valmisteluprofiilien avulla voit estää sovellusten vanhenemisen.](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
