# <a name="use-tracking-prevention-in-microsoft-edge-chromium"></a>Seurannan estämisen käyttö Microsoft Edgessä (Chromium)

Seurannan estäminen Microsoft Edgessä rajoittaa seurannan mahdollisuutta käyttää selainta ja verkkoa. Ominaisuus on suunniteltu auttamaan käyttäjiä pysymään turvassa verkossa Microsoft Edgen avulla. Lisätietoja on tiedoissa [Seurannan estäminen Microsoft Edgessä (Chromium) ja](https://go.microsoft.com/fwlink/?linkid=2135435) [selaimen tietosuoja lupaus.](https://go.microsoft.com/fwlink/?linkid=2135350)

Microsoft Edge tarjoaa kolme seurannan estämistasoa (ne voidaan valita edge://settings/privacy):

- **Basic** on vähiten rajoittava ja suunniteltu käyttäjille, jotka pitävät mukautetuista mainoksista ja jotka eivät välitä siitä, että sinua seurataan verkossa. Basic protects users only against malicious trackers, such as fingerprinters and cryptominers.
- **Tasapainotettu** on oletustaso, joka on suunniteltu käyttäjille, jotka haluavat nähdä vähemmän mainoksia, jotka seuraavat niitä verkossa. Tasapainotettu taso estää paitsi seurannat sivustoista, joita käyttäjät eivät koskaan tee, vaan myös minimoida yhteensopivuusongelmien riskin.
- **Tiukka** on rajoittavin ja suunniteltu käyttäjille, jotka eivät välitä verkkosivuston yhteensopivuudesta parhaan mahdollisen tietosuojan vuoksi.
