# <a name="manage-guest-user-access-to-planner"></a>Asiakas ohjelman käyttö oikeuden hallinta

Vieras käyttäjät voivat luoda ja poistaa säilöjä ja tehtäviä, muokata tehtävä kenttiä ja muokata paketin nimeä. Lisäksi vieras käyttäjät saavat myös joitakin ohjelmaan liittyviä ilmoituksia. Seuraavassa on joitakin usein kysyttyjä kysymyksiä Plannerin vieras käyttäjistä:

- Plannerin koonti versio on [Microsoft 365 groups-vieras käytön ominaisuus](https://support.office.com/article/Adding-guests-to-Office-365-Groups-bfc7a840-868f-4fd6-a390-f347bf51aff6). 

- Lue lisää siitä, miten voit lisätä vieras käyttäjät ohjelmaan, miten vieras käyttäjät voivat poistua työsuunnitelmaansa sekä mitä vieras käyttäjät voivat tehdä Plannerissa [Microsoft](https://support.office.com/article/Guest-access-in-Microsoft-Planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6)Plannerissa.
