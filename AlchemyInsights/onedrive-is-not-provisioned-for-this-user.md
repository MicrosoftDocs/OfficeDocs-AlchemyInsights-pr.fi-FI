# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrivea ei ole valmisteltu tälle käyttäjälle

Uusien tilien käyttäjien on kirjauduttava sisään Microsoft 365-hallinta keskukseen omalla tilillään ja käynnistettävä OneDrive-sivuston tarjoaminen manuaalisesti valitsemalla OneDrive.
  
Jos käyttäjä on jo napsauttanut OneDrivea, eikä hänen henkilökohtaista sivustoaan ole valmisteltu 24 tunnin kuluessa, kirjaudu tuki lipulla saadaksesi apua.
