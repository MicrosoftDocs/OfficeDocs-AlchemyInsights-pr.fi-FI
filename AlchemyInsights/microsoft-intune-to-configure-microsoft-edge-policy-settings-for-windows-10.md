# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Microsoft Edgen käytäntöasetusten määrittäminen Windows 10:ssä Microsoft Intunen avulla

Voit määrittää Microsoft Edgen käytäntöjä ja asetuksia lisäämällä laitemääritysprofiilin Microsoft Intuneen. Tämä vastaa Microsoft Azure Active Directory -ryhmäkäytäntöjen tai paikallisten ryhmäkäytäntöobjektien (GPO) asetusten määrittäminen käyttäjälaitteilla.

Lisätietoja on kohdassa Profiilin luominen asetusten hallintaa [varten Microsoft Edge for Windows 10:ssä.](https://go.microsoft.com/fwlink/?linkid=2133700)

Tämä artikkeli koskee Microsoft Edgen versioita 77 ja uudempia versioita.
