# <a name="move-email-to-microsoft-365"></a>Sähkö postin siirtäminen Microsoft 365

Sähkö postin siirtäminen Microsoft 365-versioon on useita vaihto ehtoja. [Microsoft 365 Mail Migration Advisor-sovelluksen](https://aka.ms/alchemyinsight-mailmigrationadvisor) avulla voit valita ympäristöösi parhaiten sopivan ja myös opastaa sinua sähkö postin siirrossa.
  
Jos haluat lukea vaihto ehdoista käynnistämättä siirtoa, Katso lisä tietoja kohdasta [siirto polun päättäminen](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Tarkista myös, että ota [käyttöön Microsoft 365 FasTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) -sivulla, jos haluat apua siirtoon, jos se on oikeutettu.
