# <a name="implement-qos-and-monitor-call-quality-in-teams"></a>QoS-ja valvo puhelujen laatua Teamsissa

Soita Analyticsia ja puhelun laadun koonti näyttöä voidaan käyttää puhelujen laatuun liittyvien ongelmien vian määritykseen Microsoft Teamsissa. Lisä tietoja palvelun laadusta (QoS), puhelun analysoinnista ja puhelun laadun koonti näytöstä (CQD) on artikkelissa [QoS-ja valvo puhelujen laatua Microsoft Teamsissa](https://docs.microsoft.com/microsoftteams/monitor-call-quality-qos).
