# <a name="about-universal-print"></a>Tietoja universaalista tulostamista

Universal Print sisältyy kaupallisiin ja oppilaitoksiin, Microsoft 365- ja Windows 10 -tilauksia, mutta se voidaan myös ostaa erillisenä tilauksena.

Opi, miten voit määrittää tulostimia, antaa käyttäjille käyttöoikeudet ja suorittaa ensimmäisen tulostustyön Pereboarding to Universal Print -sivulla.

Lisätietoja on universaalissa [käyttöoikeustulostus -| Microsoft Docs.](https://docs.microsoft.com/universal-print/fundamentals/universal-print-license)
