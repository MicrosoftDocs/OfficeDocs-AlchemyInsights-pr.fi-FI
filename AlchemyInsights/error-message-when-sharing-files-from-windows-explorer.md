# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Virhe ilmoitus, kun tiedostoja jaetaan Resurssienhallinnassa

Kun yrität jakaa Windowsin Resurssienhallinnasta, näyttöön voi tulla seuraava virhe sanoma:
  
"Valitettavasti emme pääse palvelimelle juuri nyt. Yritä myöhemmin uudelleen.
  
Jos haluat korjata virhe sanoman, valitse Internet Explorerista **Työkalut** \> **Internet-asetukset** - \> väli lehdessä **lisä asetukset** ja poista valinta kohdasta **Älä tallenna salattuja sivuja levylle**.
  
 **Huomautus**: sinun on ehkä käynnistettävä selain uudelleen, jotta asetus tulee voimaan. Lisä tietoja on kohdassa ["palvelin ei tavoita juuri nyt"-virhe sanoma tiedostojen jakamisen aikana](https://go.microsoft.com/fwlink/?linkid=2022914).
