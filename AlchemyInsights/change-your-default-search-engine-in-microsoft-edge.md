# <a name="change-your-default-search-engine-in-microsoft-edge"></a>Oletushakukoneen muuttaminen Microsoft Edgessä

Jos haluat vaihtaa oletushakukoneen, toimi seuraavasti:
1. Suorita haku osoiterivillä sen hakukoneen avulla, jonka haluat määrittää oletukseksi.
2. **Valitse... (Asetukset ja muuta) > asetukset >, haku ja palvelut.**
3. Vieritä aina Alas **Palvelut-osaan** ja valitse **osoiterivi ja hae.**
4. Valitse **haluamasi hakukone osoiterivivalikossa** käytetystä hakukoneista.
