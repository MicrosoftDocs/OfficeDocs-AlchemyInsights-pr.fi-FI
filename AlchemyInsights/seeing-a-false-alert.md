# <a name="seeing-a-false-alert"></a>Väärän ilmoituksen näkeminen

- Lähetä tiedosto Microsoftille analysoitavaksi. Lisätietoja on [ohjeaiheessa Mitä tehdä microsoft Defenderin virustentorjuntaohjelmassa väärillä positiivisilla/negatiivisilla.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis)
- Luo Salli-ilmaisin, joka estää väärän positiivisen hälytyksen. Lisätietoja on ohjeaiheessa [Tiedostojen ilmaisimien luominen](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Määritä laitteen poissulkemiset, jotta kohdetta ei tarkisteta. Lisätietoja on [ohjeaiheessa Yksittäisen Windows-laitteen poissulkemisen määrittäminen kohteen skannauksen estämiseksi](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
