# <a name="give-users-access-to-sharepoint-site"></a>SharePoint-sivuston käyttöoikeuksien antaminen käyttäjille

Käyttöoikeuspyyntöominaisuuden avulla käyttäjät voivat pyytää pääsyä sisältöön, jota heillä ei ole tällä hetkellä oikeus nähdä. Sivuston omistajana voit määrittää ominaisuuden lähettämään sinulle sähköpostia, kun joku pyytää sivuston käyttöoikeutta. Lisätietoja on kohdassa [Käyttöoikeuspyyntöjen määrittäminen ja hallinta](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

Seuraavassa on joitakin yleisiä käyttöpyyntöihin liittyviä linkkejä:

- [Käyttö estetty -luettelo tai Pyyntöhyväksyntä,kun odottava pyyntö käsitellään SharePoint Onlinessa (tämä ennen)](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Käyttöoikeuspyyntöjen asetusten muuttaminen](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Sivuston käyttöoikeuspyyntöjen hyväksyminen tai hylkääminen](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Ulkoisten käyttäjien kutsujen hallinta (vain SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Kutsun lähettäminen uudelleen](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
