# <a name="give-users-access-to-sharepoint-site"></a>SharePoint-sivuston käyttöoikeuden antaminen käyttäjille

Access request-ominaisuuden avulla käyttäjät voivat käyttää sisältöä, joka ei tällä hetkellä heillä on oikeus pyytää. Sivuston omistaja voit määrittää ominaisuuden, sähköpostin lähettämiseen, kun joku pyytää sivuston käyttämisen. Saat lisätietoja seuraavasta [Set up ja hallitse käyttöoikeuspyyntöjä](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
