# <a name="restoring-a-deleted-team-in-microsoft-teams"></a>Poistetun tiimin palauttaminen Microsoft Teamsissa

Oletko vahingossa poistanut tiimin? Jos on kulunut alle 30 päivää siitä, kun olet poistanut joukkueen, olet onnea; Voit palauttaa sen. Voit tehdä tämän noudattamalla seuraavia ohjeita: [Palauta poistettu tiimi](https://docs.microsoft.com/microsoftteams/archive-or-delete-a-team#restore-a-deleted-team).
