# <a name="create-a-team-in-microsoft-teams"></a>Tiimin luominen Microsoft Teamsissa

Voit luoda tiimin Microsoft Teamsissa seuraavasti:

1. Valitse Teamsin vasemmasta reunasta **Teams.**
2. Valitse tiimiluettelon alareunassa Liity **tai** **luo tiimi.**
3. Valitse **Luo uusi tiimi.**

Jos sinulla on ongelmia tiimin luomisessa, käytössä voi olla käytäntö. Lisätietoja on kohdassa [Microsoft 365 -ryhmien luontien hallinta.](https://docs.microsoft.com/microsoft-365/solutions/manage-creation-of-groups)
