# <a name="whats-a-hub-site"></a>Mikä on Hub-sivusto?

Hub-sivuston avulla voit hallita sivustoja ja antaa heille yleistä siirtymis-ja brändäystä. Globaalit tai SharePoint-järjestelmänvalvojat voivat luoda keskitin sivustoja ja määrittää, ketkä käyttäjät voivat yhdistää sivustonsa Hub-sivustoon. Lisä tietoja on Ohje aiheessa [SharePoint Hub-sivustot](https://go.microsoft.com/fwlink/?linkid=869388).
