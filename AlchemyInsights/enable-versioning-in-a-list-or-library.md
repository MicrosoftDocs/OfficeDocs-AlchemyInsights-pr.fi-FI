# <a name="enable-versioning-for-a-list-or-library"></a>Luettelon tai kirjaston Versiotietojen ottaminen käyttöön

1. Siirry luetteloon tai kirjastoon.
    
2. Valitse **asetukset**ja valitse sitten **kirjaston** tai **luettelon asetuksia**.
    
3. -Kohdasta **Versiotietojen asetukset**.
    
4. Valitse haluamasi asetukset ja valitse sitten **OK**.
