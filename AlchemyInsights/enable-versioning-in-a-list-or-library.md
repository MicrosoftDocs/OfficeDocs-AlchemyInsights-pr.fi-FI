# <a name="enable-versioning-for-a-list-or-library"></a>Versio tietojen ottaminen käyttöön luettelossa tai kirjastossa

1. Siirry luetteloon tai kirjastoon.
    
2. Valitse **Asetukset**ja valitse sitten **Kirjasto asetukset** tai **luettelo asetukset**.
    
3. Valitse **versio**tiedot-asetukset.
    
4. Valitse haluamasi asetukset ja valitse sitten **OK**.
