# <a name="control-sync-throughput"></a>Synkronoinnin siirtonopeuden hallinta

Jos sinun on hallittava sovellusten synkronointia, suosittelemme, että käytät verkon laatukäytäntöjä (QoS) tai Windows QoS -käytäntöjä, kun se on mahdollista. Lisätietoja verkkoasetuksista, jotka voit antaa käyttäjien käyttöön, [on ohjeaiheessa OneDrive-synkronointisovelluksen lataus- tai latausnopeuden muuttaminen.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
