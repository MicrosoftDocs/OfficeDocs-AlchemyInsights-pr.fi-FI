# <a name="remove-a-hub-site"></a>Keskitin sivuston poistaminen

Poista keskitin sivuston rekisteröinti uudessa SharePointin hallinta keskuksessa
  
1. Valitse uusi SharePoint-hallinta keskus-kohdan **sivustot**-kohdasta **Aktiiviset sivustot**.

2. Valitse sivusto, valitse **keskitin sivusto**ja napsauta sitten **Poista rekisteröityminen keskitin sivustoksi**.

3. Valitse **OK**.

[Lisä tietoja Hub-sivustoista](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
