# <a name="remove-a-hub-site"></a>Hub-sivuston poistaminen

Keskitin hallintakeskukseen uuden SharePoint-sivuston rekisteröinti
  
1. Valitse uuden SharePoint-hallintakeskukseen **sivustojen** **aktiiviset sivustot**.

2. Valitse sivusto ja valitse **pääsivusto** **Poista kuin pääsivusto**.

3. Valitse **OK**.
