# <a name="remove-a-hub-site"></a>Pääsivuston poistaminen

Voit poistaa pääsivuston rekisteröinnin uudessa SharePoint-hallinta keskuksessa seuraavasti:
  
1. Valitse uusi SharePoint-hallinta keskus-kohdan **sivustot**-kohdassa **Aktiiviset sivustot**.

2. Valitse sivusto, valitse **Hub-sivusto**ja valitse sitten **Poista rekisteröinti keskitin sivustoksi**.

3. Valitse **OK**.

[Lisä tietoja Hub-kohteista](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
