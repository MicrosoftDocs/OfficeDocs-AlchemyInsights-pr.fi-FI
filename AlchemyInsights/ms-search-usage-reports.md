# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Microsoft Search -käyttöraporttien käytön aloittaminen

Microsoft Search -käyttöraportit tarjoavat tietoja siitä, miten haku toimii organisaatiossasi. Näiden tietojen avulla voit etsiä sisältöä ja tehdä hakuista hyödyllisempiä ja miellyttävämpiä organisaatiosi käyttäjille.

Lisätietoja on kohdassa [Microsoft Search Usage Reports (Microsoft Searchin käyttöraportit).](https://go.microsoft.com/fwlink/?linkid=2152048)
