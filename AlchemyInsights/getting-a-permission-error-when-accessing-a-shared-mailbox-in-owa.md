# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Saatko käyttö oikeus virheen, kun avaat jaetulle posti laatikolle Outlookin verkko version?

Jos saat virhe ilmoituksen, **jokin meni vikaan. Sinulla ei ole oikeutta avata tätä posti laatikkoa.** Kun yrität avata jaetulle posti laatikolle Outlookin verkko versiossa, tililläsi ei ehkä ole oikeutta käyttää jaettuun posti laatikkoon. Organisaation järjestelmänvalvojasi voi määrittää tilin käyttö oikeudet jaettuun posti laatikkoon noudattamalla ohje aiheen [Luo yhteinen posti laatikko](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)-kohdan ohjeita.
