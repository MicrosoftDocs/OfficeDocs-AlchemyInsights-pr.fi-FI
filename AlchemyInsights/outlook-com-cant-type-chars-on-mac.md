# <a name="how-to-type-special-characters-on-a-mac"></a>Miten haluat kirjoittaa erikoismerkkejä Macissa

Jos haluat kirjoittaa erikoismerkkejä Mac, Avaa valikko **erikoismerkkejä** pikanäppäin ⌘ **komento** + **Control** + **tilaa**.

**ALT**on myös nopea pikakuvakkeen @ + **64**.
