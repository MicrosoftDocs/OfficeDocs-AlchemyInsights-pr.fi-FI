# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Lisätietoja Microsoft Searchin tietoturva- ja tietosuojaominaisuuksista Bingissä

Bingin Microsoft Searchin suojaus- ja tietosuojaominaisuudet auttavat organisaatiota suojaamaan käyttäjätietojaan ja työpaikkatietojaan. Pyynnöt tehdään esimerkiksi HTTPS:n kautta ja yhteydet salataan lopusta loppuun.

Lisätietoja on [Bingin Microsoft Searchin suojaus- ja tietosuojatiedoissa.](https://docs.microsoft.com/microsoftsearch/security-for-search)
