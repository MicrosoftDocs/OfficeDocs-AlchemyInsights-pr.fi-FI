# <a name="create-a-site"></a>Sivuston luominen

Voit luoda erityyppisiä SharePoint Online-sivusto tyyppejä. Lisä tietoja uuden SharePoint-hallinta keskuksen käyttämisestä on artikkelissa [sivuston luominen](https://go.microsoft.com/fwlink/?linkid=866295). Lisä tietoja käyttäjien omien sivustojen luomisesta on kohdassa [sivuston luomisen hallinta SharePoint Onlinessa](https://go.microsoft.com/fwlink/?linkid=866296).
 
Lisä tietoja uusista sivustotyypeistä on Ohje aiheessa [yhdistettyjen ryhmä sivustojen](https://go.microsoft.com/fwlink/?linkid=866292) tai [viestintä sivustojen](https://go.microsoft.com/fwlink/?linkid=866294)ryhmitteleminen.
