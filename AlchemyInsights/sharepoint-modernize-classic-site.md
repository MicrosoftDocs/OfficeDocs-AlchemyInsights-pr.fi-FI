# <a name="modernize-classic-sharepoint-site"></a>Ajanmukaistaa classic SharePoint-sivustoon

Tiedonvaihdossa tarvittavia perinteinen, Moderni kokemus SharePoint-sivustoja vaiheittaisia katsauksen on [perinteinen SharePoint-sivustojen Modernize](https://docs.microsoft.com/en-us/sharepoint/dev/transform/modernize-classic-sites).

Voit maksimoida käyttö sivuston ja luettelon ja kirjaston sivujen Moderni käyttöliittymä, katso [käyttöliittymä Modernize](https://docs.microsoft.com/en-us/sharepoint/dev/transform/modernize-userinterface). 

Lisätietoja luetteloiden ja kirjastojen salliminen käytettäessä Moderni käyttöliittymä on [Moderni luetteloiden ja kirjastojen avulla suurennus](https://docs.microsoft.com/en-us/sharepoint/dev/transform/modernize-userinterface-lists-and-libraries)

Saat lisätietoja Office 365-ryhmän muodostaminen ja ryhmän yhteys suorittamista [Office 365-ryhmän muodostaminen](https://docs.microsoft.com/en-us/sharepoint/dev/transform/modernize-connect-to-office365-group).
