# <a name="add-users-to-a-sharepoint-group"></a>Käyttäjien lisääminen SharePoint-ryhmään

SharePoint-ryhmä on kokoelma käyttäjiä, joilla kaikilla on samat käyttö oikeudet sivustolle ja sisällölle. Käyttö oikeuksien määrittäminen yhdelle henkilölle kerrallaan voit määrittää saman käyttö oikeus tason kätevästi useille henkilöille yhtä aikaa ryhmien avulla.

- [Käyttäjien lisääminen ryhmään](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Jäsenten lisääminen tai poistaminen Microsoft 365-ryhmistä hallinta keskuksen avulla](https://docs.microsoft.com/microsoft-365/admin/create-groups/add-or-remove-members-from-groups)
