# <a name="microsoft-stream-external-sharing"></a>Microsoft Streamin ulkoinen jakaminen

Microsoft Stream ei vielä tue ulkoisia jakamisskenaarioita organisaation ulkopuolisille vieraille tai henkilöille. Tiedämme, että näitä ominaisuuksia tarvitaan erittäin paljon, ja pyrimme laatimaan suunnitelmia näiden ominaisuuksien rakentamiseksi.
