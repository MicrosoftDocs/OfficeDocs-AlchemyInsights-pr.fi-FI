# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Haettaessa lupaa virhe avattaessa jaetun Exchange-postilaatikon Outlook Web?

Virhe **tapahtuiko virheitä. Sinulla ei ole oikeuksia tämän postilaatikon avaamiseen.** Kun yrität avata jaetun Exchange-postilaatikon Outlook Web-tilisi ei ehkä ole oikeutta käyttää jaetun postilaatikon. Organisaation järjestelmänvalvoja voi määritellä oman tilin käyttöoikeudet jaetun postilaatikon noudattamalla ohjeaiheen [jaetun postilaatikon luominen](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
