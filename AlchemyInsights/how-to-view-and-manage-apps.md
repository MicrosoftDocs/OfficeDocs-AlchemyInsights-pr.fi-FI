# <a name="how-to-view-and-manage-apps"></a>Miten voit tarkastella ja hallita apps

Voit tarkastella oman apps, on oltava uusi admin Centerissä.  Voit vaihtaa admin Centerin kotisivulla.  

1. Uusi admin Centerin käytössä, siirry **laskutuksen** > [tuotteet & palvelut](https://go.microsoft.com/fwlink/p/?linkid=842054) sivulta.

2. **_AMP_ tuotteet-palvelut** -sivulla Valitse **Apps** -välilehti.
