# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Useita aktiivisia istuntoja samaan postilaatikkoon

Exchange-resurssien käytön hallitsemiseksi postilaatikolla on budjetti.

Budjetin yli -poikkeus voidaan käynnistää seuraavissa tilanteissa, mutta ei rajoitu siihen:

- Samassa Outlook Web App -istunnossa avataan muutamia selaimen välilehtiä.

- Muutama aktiivinen Outlook Web App -istunto samaan postilaatikkoon.

- Muutamat muut asiakassovellukset (Outlook, Outlook Mobile, kolmannen osapuolen asiakassovellus) voivat käyttää postilaatikkoa samanaikaisesti.

- Pitkäaikaiset toiminnot, kuten hakupyyntöjen suorittaminen, suoritetaan toisesta aktiivisesta postilaatikkoistunnosta.
