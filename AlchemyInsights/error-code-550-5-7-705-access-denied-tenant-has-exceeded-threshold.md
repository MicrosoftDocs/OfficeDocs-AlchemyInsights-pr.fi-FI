# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 on estetty, vuokralaisen on ylittänyt raja-arvon

Tämä tapahtuu, kun Office 365-organisaatiosi on lähetetty liian paljon roskapostia tai bulk mail. Jos haluat poistaa tämän lohkon, täytyy ymmärtää ja selittää syy tukea agentti, sekä korjata ongelman.
