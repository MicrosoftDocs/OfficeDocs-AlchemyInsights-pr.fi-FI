# <a name="how-to-stop-signing-in-automatically"></a>Miten voit lopettaa automaattinen kirjautuminen

Lopeta automaattinen kirjautuminen-valintaruudun valinta, **Pysy kirjautuneena sisään** sivulla, johon kirjoitat salasanan. 
  
Lisätietoja on kohdassa [miten kirjautua sisään, tai pois Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873113).
