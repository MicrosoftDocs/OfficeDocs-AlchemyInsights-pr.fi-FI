# <a name="hybrid-configuration-wizard"></a>Ohjattu yhdistelmämääritystoiminto

Ohjattu yhdistelmämääritystoiminto on sovellus, jonka avulla voit määrittää uuden yhdistelmäkäyttöönoton paikallisen Exchange-organisaation ja Microsoft 365:n välille. Jos yhdistelmäkäyttöönotto on jo määritetty, ohjattu toiminto auttaa päivittämään yhdistelmäkäyttöönottoasetukset. Lisätietoja:
  
- [Ohjattu yhdistelmämääritystoiminto](https://technet.microsoft.com/library/mt595788%28v=exchg.150%29.aspx)

- [Ohjattu yhdistelmämääritystoiminto](https://technet.microsoft.com/library/hh529921%28v=exchg.150%29.aspx)

- [Ohjatun yhdistelmämääritystoiminnon usein kysytyt usein kysytyt kohteet](https://technet.microsoft.com/library/mt488940%28v=exchg.150%29.aspx)
