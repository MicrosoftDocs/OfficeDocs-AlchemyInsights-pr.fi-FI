# <a name="need-to-create-an-email-signature"></a>Luo sähköposti-allekirjoitusta tarvitaan?

Jos sinulla on tili Outlook tai Outlook web (Office 365-työ tai Koulu)-tilin, voit allekirjoituksen luominen ja se lisätään automaattisesti kaikkiin lähteviin viesteihin tai lisätä valikoivasti tietyt lähteviin viesteihin. Huomaa, että sähköpostin allekirjoitus on eri Outlookissa ja Outlook web-sivulla. Luo sähköpostin allekirjoitus-kohdassa [Luo allekirjoituksen lisääminen Outlook-viesteissä ja](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) tai [Luo ja lisätä allekirjoituksen sähköposti Outlookissa, web-](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
