# <a name="change-your-pin"></a>PIN-koodin vaihtaminen

Outlook.com ei käytä PIN-koodia, mutta voit kirjautua tietokoneeseen PIN-koodilla tai Windows Hellolla. PIN-koodin vaihtaminen Windows 10:ssä:

1. Valitse Aloitus-painike tai paina näppäimistön**Windows-näppäintä**ja valitse sitten **Asetukset** > **Tilien** > kirjautumisasetukset .
2. Valitse **PIN -kohdassa** **Vaihda** -painike.
3. Anna nykyinen PIN-koodisi ja anna uusi PIN-koodisi alla.

[Lue lisää Windows Hellosta.](https://support.microsoft.com/help/17215/)
