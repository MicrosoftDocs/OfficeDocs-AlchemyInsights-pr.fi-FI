# <a name="create-and-use-a-shared-mailbox"></a>Luoda ja käyttää jaetun postilaatikon

Jaetun postilaatikon antaa useammalle kuin yhdelle henkilölle, lähettää ja vastaanottaa sähköposti tunnus, kuten **sales@contoso.com**, mutta uusi käyttöoikeus.
  
[Jaetun postilaatikon luominen](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) kuvataan, miten: 
  
- Jaetun postilaatikon luominen
    
- Antaa käyttäjille oikeudet käyttää jaettua postilaatikkoa
    
- Jaetun Exchange-postilaatikon Outlook- tai Outlook Web Access
