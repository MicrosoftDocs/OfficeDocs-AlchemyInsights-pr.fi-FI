# <a name="email-aliases-on-mobile-devices"></a>Sähköpostialiakset mobiililaitteissa

Sähköpostialias on vain yksi postilaatikkoon määritetty sähköpostiosoite, joten käyttäjän aliakseen lähetetyt viestit toimitetaan heidän postilaatikkoonsa ja ne ovat luettavissa mobiililaitteellaan. Lisämäärityksiä ei tarvita.

Käyttäjä ei kuitenkaan voi vastata viesteihin tai lähettää uusia viestejä aliaksella From-osoitteenaan. Kaikki lähtevät viestit tulevat heidän ensisijaisesta sähköpostiosoitteestaan.
