# <a name="export-user-account-information"></a>Käyttäjätilin tietojen vieminen

Jos haluat viedä käyttäjätilitietoja Microsoft 365 -hallintakeskuksessa, mukaan lukien tiedot käyttöoikeuksista, salasanakäytännöstä, estetyt tilit, AAD Sync -tila ja paljon muuta, siirry **Käyttäjät** > [aktiiviset käyttäjät -sivulle](https://go.microsoft.com/fwlink/p/?linkid=834822) ja valitse sitten **Vie käyttäjät**.
