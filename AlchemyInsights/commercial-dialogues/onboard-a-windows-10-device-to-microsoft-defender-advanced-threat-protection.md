# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Windows 10 -laitteen käynnistäminen Microsoft Defender Advanced Threat Protectioniin (MDATP)

MDATP tukee seuraavia Windows 10:n käyttöönottotyökaluja ja -menetelmiä:

- Ryhmäkäytäntö
- Microsoft Endpoint Configuration Manager
- Mobiililaitteiden hallinta, mukaan lukien Microsoft Intune
- Paikalliset komentosarjat

Lisätietoja on ohjeaiheessa [Windows 10 -laitteiden pereboardointityökalut ja -menetelmät.](https://go.microsoft.com/fwlink/?linkid=2143460)
