# <a name="fix-problem-viewing-protected-message-in-outlook"></a>Suojatun viestin näyttämisen ongelmien ratkaiseminen Outlookissa

Joissain tapauksissa Outlook ei ehkä voi avata toisen Microsoft 365 -organisaation käyttäjältä lähetettyä viestiä, jos viesti on suojattu oikeuksien hallinnassa.

Voit avata ja tarkastella suojattuja viestejä seuraavasti:

1. Varmista, että olet asentanut Outlook 1808:n tai uudemman version. Jos haluat selvittää versionumerosi, katso [mitä Office-versiota käytössäni on?](https://support.microsoft.com/office/about-office-what-version-of-office-am-i-using-932788b8-a3ce-44bf-bb09-e334518b8b19)
2. Kokeile tarkastella sähköpostia Outlookin verkkosovelluksessa: [Outlookin verkkosovelluksen sisäänkirjautuminen](https://outlook.office365.com/mail/inbox)

> [!NOTE]
> Ongelmaa korjataan Outlook 2016:ssa ja Outlook 2019:ssä. Sitä ei korjata Outlook 2010:ssä tai 2013:ssa.
