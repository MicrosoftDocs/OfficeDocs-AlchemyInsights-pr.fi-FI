# <a name="report-suspicious-urls"></a>Epäilyttävistä URL-osoitteista ilmoittaminen

Voit lähettää epäilyttäviä URL-osoitteita seuraavasti:

1. Kirjoita Uusi **lähetys -pikaikkunaan** koko URL-osoite, mukaan lukien protokolla (https://).
2. Jos valitset Pitäisi **olla estetty, määritä,** onko URL-osoite tietojen kalastelu **vai** **haittaohjelma.**
3. Valitse **Lähetä.**
