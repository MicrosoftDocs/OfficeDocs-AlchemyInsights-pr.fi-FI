# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Windows-palvelimen käynnistäminen Microsoft Defender for Endpointiin

Windows Server -käyttöjärjestelmässä Microsoft Defender for Endpoint tarjoaa kehittyneitä ominaisuuksia hyökkäysten tunnistamiseen ja tutkimiseen. Nämä ominaisuudet tarjotaan saumattomasti Microsoft Defender Security Center -konsolin kautta.

Defender for Endpoint tukee seuraavia Windows Server -versioita:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) -versio 1803 ja uudemmat versiot
- Windows Server 2019 ja uudemmat
- Windows Server 2019 Core Edition

Lisätietoja on ohjeaiheessa [Onboard Windows -palvelimet Microsoft Defender for Endpoint -palveluun.](https://go.microsoft.com/fwlink/?linkid=2143627)
