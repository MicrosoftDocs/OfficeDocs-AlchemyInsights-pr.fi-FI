# <a name="assign-ediscovery-manager-permissions"></a>eDiscovery Manager -käyttöoikeuksien määrittäminen

Voit määrittää eDiscovery Manager -käyttöoikeudet seuraavasti:

1. Valitse [Office 365:n &](https://sip.protection.office.com/)yhteensopivuuskeskuksessa **eDiscovery Managerin > käyttöoikeudet.**
2. Valitse **eDiscovery Manager** -ruudussa **eDiscovery Managerin vieressä** **Muokkaa.**
3. Valitse Muokkaa **eDiscovery Manager -ruudussa** **Valitse eDiscovery Manager.**
4. Valitse **Valitse eDiscovery-hallinta** -ruudussa **+ Lisää** ja valitse sitten käyttäjien nimet.
5. Valitse **Lisää > Valmis > Tallenna.**
