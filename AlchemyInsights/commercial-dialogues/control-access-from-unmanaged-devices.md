# <a name="control-the-access-from-unmanaged-devices"></a>Hallitsemattomien laitteiden käytön hallinta

Microsoft 365:n SharePoint-järjestelmänvalvojana tai yleisenä järjestelmänvalvojana voit estää SharePoint- ja OneDrive-sisällön käytön hallitsemattomista laitteista tai rajoittaa niiden käyttörajoitusta. Lisätietoja on kohdassa [Hallitsemattomien laitteiden käytön hallinta.](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
