# <a name="workflows-in-sharepoint"></a>Työn kulut SharePointissa

SharePointin työn kulkujen avulla voit mallintaa ja automatisoida liiketoiminta prosesseja. SharePoint tukee kahta päätyön kulun kehitys ympäristöä työn kulkujen luomiseen: SharePoint Designer ja Visual Studio. 

Lisä tietoja on kohdassa SharePoint- [työn kulkujen kehittäminen SharePoint Designerissa ja Visiossa](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) tai [SharePoint-työn kulkujen kehittäminen Visual Studion avulla](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) kunkin edun ja haittojen arvioimiseksi. 

Jos yrität selvittää, onko työn kulut ratkaisu SharePoint-ympäristössä tai haluatko yleisiä tietoja käytön aloittamisesta, tutustu Ohje aiheeseen [työn kulkujen yleiskatsaus SharePointissa](https://docs.microsoft.com/sharepoint/dev/general-development/get-started-with-workflows-in-sharepoint#overview-of-workflows-in-sharepoint).
