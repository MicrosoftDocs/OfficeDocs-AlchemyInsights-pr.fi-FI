# <a name="print-results-in-a-summary-of-responses"></a>Tulosta tulokset yhteenvedon vasta uksista

Voit [tulostaa Microsoft Forms-tulokset](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) yhteenvedon vastausten tai yksittäisen henkilön tuloksista. 

Et voi tulostaa lomakkeita ilman vasta uksia, mutta voit pyytää tai äänestää tästä ominaisuudesta vierailemalla [Microsoft Forms UserVoice-sivustossa](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
