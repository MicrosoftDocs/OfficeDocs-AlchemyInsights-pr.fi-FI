# <a name="print-results-in-a-summary-of-responses"></a>Tulosta yhteenveto vastauksista-tulokset

Voit [tulostaa lomakkeita Microsoft-tuloksia](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) kuin yhteenveto vastauksista tai yksittäisen. 

Lomakkeiden vastaukset ilman ei voi tulostaa, mutta voit pyytää tai äänestää tätä toimintoa [Microsoft Forms UserVoice sivusto](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
