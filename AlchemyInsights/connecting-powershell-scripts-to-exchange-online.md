# <a name="connecting-powershell-scripts-to-exchange-online"></a>PowerShell-komentosarjojen yhdistäminen Exchange Onlineen

Exchange Onlinen perustodennus poistetaan käytöstä, ja jatkotapa on muodostaa yhteys käyttämällä komentosarjojen ja valvomattomien tehtävien varmennepohjaista todennusta. Lisätietoja on [EXO V2 -moduulin kohdassa Vain app only -todennus automaattisia komentosarjoja varten](https://docs.microsoft.com/powershell/exchange/app-only-auth-powershell-v2).
