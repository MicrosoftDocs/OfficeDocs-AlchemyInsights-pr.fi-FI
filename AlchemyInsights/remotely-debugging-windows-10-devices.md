# <a name="get-started-with-remotely-debugging-windows-10-devices"></a>Windows 10-laitteiden etävianmäärityksen aloittaminen

Windows 10-laitteen Windows-tai macOS-tieto koneen Live-sisällön vian määritys etäyhteyden kautta:

1. Määritä Windows 10-laitteesi etävirheen korjausta varten ja muodosta yhteys siihen kehitys koneesta.
2. Tutki ja debug Live-sisältöä Windows 10-laitteessasi kehitys koneesta.
3. Screencast sisältöä Windows 10-laitteesta kehitys koneen DevTools-esiintymään.

Lisä tietoja on artikkelissa [Windows 10-laitteiden etävianmäärityksen aloittaminen Microsoft Edgellä](https://go.microsoft.com/fwlink/?linkid=2142172).
