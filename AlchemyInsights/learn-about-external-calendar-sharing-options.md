# <a name="exchange-online-collaboration-options"></a>Exchange Onlinen yhteiskäyttöasetukset

- Jos haluat **yleiskatsauksen ulkoisesta kalenterin jakamisesta**, Katso lisä tietoja artikkelista [jakaminen Exchange Onlinessa](https://technet.microsoft.com/library/jj916670%28v=exchg.150%29.aspx).

- Jos haluat **jakaa kalentereita toisen Office 365-organisaation kanssa**, Katso lisä tietoja artikkelista [organisaation suhteet Exchange Onlinessa](https://technet.microsoft.com/library/jj916658%28v=exchg.150%29.aspx).

- Jos haluat hallita sitä, **miten käyttäjät jakavat kalentereitaan organisaatiosi ulkopuolisten henkilöiden kanssa**, Katso lisä tietoja artikkelista [käytäntöjen jakaminen Exchange Onlinessa](https://technet.microsoft.com/library/jj916673%28v=exchg.150%29.aspx).

- Jos haluat nähdä kalenterin jakamisen **parannukset** , tutustu Ohje aiheeseen [kalenterin jakaminen Microsoft 365](https://support.office.com/article/calendar-sharing-in-microsoft-365-b576ecc3-0945-4d75-85f1-5efafb8a37b4)-sovelluksessa.
