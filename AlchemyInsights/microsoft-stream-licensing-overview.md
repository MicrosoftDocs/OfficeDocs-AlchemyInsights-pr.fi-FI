# <a name="microsoft-stream-licensing-overview"></a>Microsoft Stream -käyttöoikeuksien yleiskatsaus

Microsoft Stream on pilvipalvelu, jolla on käyttöoikeus käyttäjän tilauskohtaisesti. Se on saatavana osana kelvollisia Office 365 -tilauksia. Lisätietoja kustakin tilaustyypistä on Kohdassa Office 365 pricing.
