# <a name="use-microsoft-graph-api-to-manage-groups"></a>Ryhmien hallinta Microsoft Graph API:n avulla

Ryhmät ovat käyttäjäkokoelmia ja muita pääkäyttäjiä, jotka jakavat resurssien käyttöoikeudet Microsoftin palveluissa tai sovelluksessa. Microsoft Graphissa on ohjelmointirajapinnat, joita voit käyttää erilaisten ryhmä- ja ryhmätoimintojen luomiseen ja hallintaan skenaarion mukaan. Kaikki ryhmään liittyvät toiminnot Microsoft Graphissa edellyttävät järjestelmänvalvojan suostumusta. Lisätietoja on kohdassa [Ryhmien käyttö Microsoft Graphissa.](https://docs.microsoft.com/graph/api/resources/groups-overview)
