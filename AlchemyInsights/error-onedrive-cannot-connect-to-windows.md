# <a name="error-onedrive-cannot-connect-to-windows"></a>Virhe: OneDrive ei voi muodostaa yhteyttä Windows

Voit korjata virhesanoman, Etsi cldflt.sys tiedoston Katso jos se toimii, ja jos ei ole, ota se käyttöön. 
  
Saat lisätietoja siitä, kuinka voit ottaa käyttöön cldflt.sys-tiedostoa [OneDrive ei voi muodostaa yhteyttä Windows-virhe käytettäessä tiedostoja OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
