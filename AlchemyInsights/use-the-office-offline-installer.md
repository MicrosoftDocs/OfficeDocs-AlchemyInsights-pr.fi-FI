# <a name="use-the-office-offline-installer"></a>Officen offline-asennusohjelman käyttäminen

Sinun on oltava yhteydessä Internetiin, jotta voit ladata asennustiedostot, mutta kun se on tehty, voit asentaa Officen offline-tilaan tietokoneeseen kätevästi Officedeployment Tool (ODT) -työkalun avulla.

ODT on suunniteltu yritysympäristöihin ja toimii komentoriviltä, joten vaiheet ovat monimutkaisempia - mutta ne toimivat silti asennettavaksi yhteen laitteeseen. Katso [Officen offline-asennusohjelman käyttäminen](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA).
