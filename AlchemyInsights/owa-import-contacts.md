# <a name="import-contacts"></a>Yhteystietojen tuominen

Yhteystietojen tuominen Outlookin web-sivulla, sinun on viedä muita email-palvelun, .csv-tiedostona. Noudata alla olevia ohjeita.

1. Valitse sivun alareunassa ihmiset kuvaketta siirtymällä henkilöt-sivulle.
2. Valitse työkaluriviltä **Manage** > **Tuo yhteystiedot**.
3. Valitse **Selaa**, valitse CSV-tiedosto ja valitse sitten **Avaa**.
4. Valitse **Tuo**.

Lisätietoja on kohdassa [Outlook tuo yhteystiedot](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
