# <a name="create-and-use-a-shared-mailbox"></a>Luoda ja käyttää jaetun postilaatikon

Jaetun postilaatikon antaa useammalle kuin yhdelle henkilölle, lähettää ja vastaanottaa sähköposti tunnus, kuten sales@contoso.com, mutta uusi käyttöoikeus.
  
- [Voit aloittaa](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

Voit myös lukea [jaetun postilaatikon luominen](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) artikkelin joka kertoo kaiken tarvittavan tietää jaettujen postilaatikoiden, mukaan lukien niiden käyttämisestä.
