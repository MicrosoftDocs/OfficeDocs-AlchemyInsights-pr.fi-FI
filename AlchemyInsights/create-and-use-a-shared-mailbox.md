# <a name="create-and-use-a-shared-mailbox"></a>Luoda ja käyttää jaetun postilaatikon

Jaetun postilaatikon antaa useammalle kuin yhdelle henkilölle, lähettää ja vastaanottaa sähköposti tunnus, kuten sales@contoso.com, mutta uusi käyttöoikeus.
  
- [Tekemällä seuraavat toimet](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) Microsoft 365 admin Centerissä aloittamiseen. 

Voit myös lukea artikkelin [jaetun postilaatikon luominen](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) , joka kertoo kaiken tarvittavan tietää jaettujen postilaatikoiden, mukaan lukien niiden käyttämisestä.
