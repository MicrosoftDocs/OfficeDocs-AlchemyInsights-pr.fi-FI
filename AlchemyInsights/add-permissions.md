# <a name="add-permissions"></a>Käyttöoikeuksien lisääminen

1. Järjestelmänvalvojan suostumuksen päätepisteen avulla voit määrittää [sovelluksen staattisesti pyydettyjen käyttöoikeuksien luettelon.](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application)
1. Voit myöntää mahdollisuuden luoda sovellusrekisteröinnit kahdella eri tavalla. Lisätietoja on kohdassa [Azure Active Directoryn mukautettujen roolien sovellusten rekisteröintioikeudet.](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions)
1. [Roolin käyttöoikeudet:](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions)taulukot, jotka kuvaavat kullekin roolille annettuja Azure Active Directoryn käyttöoikeuksia. Joillakin rooleilla voi olla lisäkäyttöoikeuksia Microsoft-palveluissa Azure Active Directoryn ulkopuolella.
