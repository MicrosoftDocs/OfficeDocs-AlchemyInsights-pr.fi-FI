# <a name="cant-install-or-run-the-ediscovery-export-tool"></a>Ei voi asentaa tai suorittaa eDiscovery vientityökalu?

Jos et voi asentaa tai suorittaa Office 365-eDiscovery vientityökalu Lataa hakutulokset, tarkista seuraavat asiat:
  
- Käyttämäsi tietokone täyttää nämä vaatimukset:
    
  - 32 - tai 64-bittiset versiot Windows 7 ja uudemmat versiot
    
  - Microsoft .NET Frameworkin 4.7
    
  - Tuettu selain:
    
  - Microsoft Edge
    
    Tai
    
  - Internet Explorer 10 ja uudemmat versiot
    
    Muissa selaimissa, kuten Google Chrome ja Mozilla Firefox ei tue.
    
- Organisaatiosi voi muodostaa päätepiste, joka on Azure- ** \*. blob.core.windows.net** (yleismerkkejä edustaa vientityö yksilöllinen). 
    
- Sinulle on määritetty Office 365-suojauksen rooli Vie &amp; Compliance Centeriin. Oletusarvoisesti tämä rooli määritetään vain eDiscovery hallinta-rooliryhmään. Katso [eDiscovery käyttöoikeuksien määrittäminen](https://support.office.com/article/assign-ediscovery-permissions-in-the-office-365-security-compliance-center-5b9a067b-9d2e-4aa5-bb33-99d8c0d0b5d7#moreinfo).
    
Lisätietoja [Vie sisällön hakutuloksista](https://support.office.com/article/Export-Content-Search-results-from-the-Office-365-Security-Compliance-Center-ed48d448-3714-4c42-85f5-10f75f6a4278).
