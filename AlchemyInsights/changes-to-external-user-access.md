# <a name="changes-to-external-user-access"></a>Ulkoiseen käyttö oikeus tiedostoon tehdyt muutokset

Maaliskuun 23, 2018, ulkoiset käyttäjät eivät voi enää käyttää sisältöä, joka on jaettu kaikille henkilöille, kaikille todennetuille käyttäjille tai kaikille lomakkeiden käyttäjille, vaikka heidät olisi kutsuttu tai lisätty valtuutettuihin ryhmiin. Ulkoiset käyttäjät voivat käyttää vain sisältöä, joka on jaettu käyttäjälle tai ryhmiin, joihin käyttäjä kuuluu. Jos haluat säilyttää nykyiset asetuksesi, Lue seuraavat: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
