# <a name="changes-to-external-user-access"></a>Muutokset ulkoiseen käyttöön

23.3.2018 alkaen ulkoiset käyttäjät eivät voi enää käyttää sisältöä, joka on jaettu kaikkien, kaikkien todennettujen käyttäjien tai kaikkien lomakkeiden käyttäjien kanssa, vaikka heidät olisi kutsuttu tai lisätty valtuutettuihin ryhmiin. Ulkoiset käyttäjät voivat käyttää vain sisältöä, joka on jaettu käyttäjän kanssa tai ryhmien kanssa, joihin käyttäjä kuuluu. Jos haluat säilyttää nykyiset asetuksesi, katso:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
