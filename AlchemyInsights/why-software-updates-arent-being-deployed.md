# <a name="why-software-updates-are-not-being-deployed"></a>Miksi ohjelmisto päivityksiä ei ole otettu käyttöön?

Tarkista seuraavat asiat, jos olet määrittänyt ohjelmisto päivitykset Windows 10: n päivitys renkaalla, mutta päivityksiä ei ole otettu käyttöön.  

- Harkitse Windows-ylläpidon vaihtamista  **puolivuosittaisista kanava**  versiosta tiukempaan, useammin julkaistavan julkaisun tyyppiin  
- Tarkista  **laatu päivityksen**  ja  **ominaisuuksien päivityksen**lykkäys vaihe. Lykkäys aika voi johtaa päivitysten viivästymiseen jopa 180 päivän ajan.
