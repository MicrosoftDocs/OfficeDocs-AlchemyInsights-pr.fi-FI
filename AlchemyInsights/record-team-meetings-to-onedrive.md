# <a name="teams-meeting-recordings-to-onedrive"></a>Teamsin Kokous tallenteet OneDriveen

Microsoft Teamsissa on uusi menetelmä Kokous tallenteiden tallentamista varten. Tämä menetelmä tallentaa Microsoft OneDrive for Business-ja SharePoint-tallennukset Microsoft 365-palveluun ja tarjoaa monia etuja, koska se on ensimmäinen vaihe siirtymisessä klassisesta Microsoft Streamista [uuteen virtaan](https://docs.microsoft.com/stream/streamnew/new-stream).  

- [Koko uksen tallennus vaihtoehdon määrittäminen OneDrive for Businessiin ja SharePointiin](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint)
- [OneDrive for Businessin ja SharePointin käytöstä poistaminen, jotta voit jatkaa Streamista](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#opt-out-of-onedrive-for-business-and-sharepoint-to-continue-using-stream)  
- [Usein kysytyt kysymykset](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#frequently-asked-questions)
