# <a name="import-contacts"></a>Yhteystietojen tuominen

Tarvitset tuomalla Outlook.com yhteystiedot viedään muita email-palvelun, .csv-tiedostona. Noudata alla olevia ohjeita.
  
1. Siirry [ihmisiä](https://outlook.live.com/people/).

2. Valitse sivun yläosassa **Hallitse** \> **Tuo yhteystiedot**.

3. Valitse **Selaa**, siirry kansioon, johon viedyt yhteystiedot tallennetaan ja valitse **Avaa**.

Lisätietoja on kohdassa [Tuo yhteystiedot Outlook.com-tilille](https://go.microsoft.com/fwlink/p/?linkid=873136).
