# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Microsoft Defender ATP ei huomannut haitallista kohdetta

- Lähetä haitallinen tiedosto Microsoftille analysoitavaksi [Microsoft Defender Response Portalissa](https://www.microsoft.com/wdsi/filesubmission/). 
- Jos olet kirjautunut sisään, etsi tiedoston lähetyshistoria [Microsoft Defender Response Portalista](https://www.microsoft.com/wdsi/submissionhistory).
- Lisätietoja on kohdassa [Tiedostojen lähettäminen analysointia varten](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
