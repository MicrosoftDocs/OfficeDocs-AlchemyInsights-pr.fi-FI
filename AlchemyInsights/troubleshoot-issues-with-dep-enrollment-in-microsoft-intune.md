# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Dep-rekisteröinnin ongelmien vianmääritys Microsoft Intunessa

Ratkaise ongelmasi nyt alla lueteltujen resurssien avulla.
  
1. Jos DEP-laite ei pysty rekisteröimään ja monimenetelmäinen todentamismenetelmä (Multi-Factor Authentication) on käytössä, poista monimenetelmäinen todentamismenetelmä käytöstä. Tällä hetkellä MFA:ta ei tueta DEP-rekisteröinnissä

2. [Intunen vianmääritysportaalin avulla](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) voit diagnosoida ja ratkaista yleisiä rekisteröintivirheitä. Katso [lisätietoja tästä](https://docs.microsoft.com/intune/help-desk-operators) asiakirjasta.

3. Tarkista nämä asiakirjat ja luettelo yleisistä virheistä, jotka estävät kunkin rekisteröinnin ja ratkaisut: [Vianmääritysopas](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) ja [Vianmääritys-asiakirja](https://docs.microsoft.com/troubleshoot/mem/intune/troubleshoot-device-enrollment-in-intune)

4. [Lisätietoja laitteen rekisteröintiohjelmasta.](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios)
