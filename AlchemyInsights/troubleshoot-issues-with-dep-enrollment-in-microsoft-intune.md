# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Tietojen suorittamisen estämisen rekisteröintiin liittyvien ongelmien vianmääritys Microsoft Intunen

Ratkaise ongelma nyt alla olevien resurssien avulla.
  
1. Jos tietojen suorittamisen estäminen ei onnistu ja MFA (Multi-Factor Authentication) on käytössä, poista mfa käytöstä. Tällä hetkellä tietojen suorittamisen estämisen rekisteröintiei tue mfa-arvoa

2. [Intune Troubleshooting Portalin](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) avulla voit diagnosoida ja ratkaista yleisiä rekisteröintivirheitä. Lisätietoja on [tässä asiakirjassa.](https://docs.microsoft.com/intune/help-desk-operators)

3. Näiden asiakirjojen avulla on luettelo yleisistä virheistä, jotka estävät rekisteröitymisen ja tarkkuuden kuhunkin: [Vianmääritysopas](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) ja [Vianmääritys-asiakirja](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)

4. [Lisätietoja laitteen rekisteröintiohjelmasta](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios).
