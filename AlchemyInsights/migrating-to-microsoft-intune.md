# <a name="migrating-to-microsoft-intune"></a>Siirtyminen Microsoft Intuneen

Jos yrität siirtyä Intuneen, tarkista [Intune-siirto opas](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Tarkempia ohjeita System Center Configuration Manageriesta (ja SCCM-risteytymistä) Intune-järjestelmään on kohdassa [hybridin siirto asiakirja](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
