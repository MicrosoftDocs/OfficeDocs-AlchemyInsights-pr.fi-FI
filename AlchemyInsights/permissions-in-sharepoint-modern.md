# <a name="permissions-in-the-sharepoint-modern-experience"></a>SharePointin modernin käyttö kokemuksen oikeudet

Perinteisesti SharePointin käyttö oikeuksia on hallinnoitu sivuston käyttö oikeus ryhmien avulla (omistajat, jäsenet, vierailijat jne.). SharePointissa Microsoft 365-sovelluksessa tämä pysyy voimassa tietyntyyppisissä kohteissa, mutta käytettävissä on muitakin vaihto ehtoja.  

Lisä tietoja on [kohdassa SharePointin moderni käyttö kokemus](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions).
