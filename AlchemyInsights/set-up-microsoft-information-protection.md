# <a name="set-up-microsoft-information-protection"></a>Microsoft Information Protectionin määrittäminen

Saat yleiskatsauksen ominaisuuksista, joita voit käyttää tieto suoja strategiassa, jotta voit olla varma, että luottamukselliset tietosi on suojattu. Käytä neljän vaiheen elin kaari ajattelua, jossa voit etsiä, luokitella, suojata ja valvoa arkaluonteisia tietoja. Microsoft Information Protectionin asennus opas sisältää ohjeita kunkin vaiheen suorittamiseen.
