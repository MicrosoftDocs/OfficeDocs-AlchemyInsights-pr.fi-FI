# <a name="proxyaddress-incorrect"></a>ProxyAddress virheellinen

Kun objekti synkronoidaan Azure adiin, Active Directoryn Proxyosoitteet-määritteeseen määritettyjä arvoja verrataan Azure AD-sään töihin, ja Proxyosoitteet-määrite täytetään Azure ADISSÄ. Tämän vuoksi Active Directoryn objektin Proxyosoitteet-määritteen arvot eivät välttämättä ole samat kuin Azure ADIN Proxyosoitteet-määritteen arvot.
  
Lisä tietoja proxyAddress-ominaisuuden täyttymistä on kohdassa [Miten proxyAddress-määrite täytetään Azure adissa](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
