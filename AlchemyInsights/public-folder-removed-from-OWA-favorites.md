# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Internet-Outlook (OWA) -suosikeista poistettu yleinen kansio

OWA:n suosikeista poistetuissa julkisissa kansioissa ilmenee tunnettu ongelma jokaisen kirjautumisen tai näytön päivityksen jälkeen. Ongelma korjataan pian. Lisää siihen asti kansio uudelleen OWA:n käytön ajaksi.
