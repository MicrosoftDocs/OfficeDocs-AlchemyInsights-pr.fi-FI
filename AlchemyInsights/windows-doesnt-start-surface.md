# <a name="windows-doesnt-start-on-surface"></a>Windows ei käynnisty Surfacessa

Jos Windows ei käynnisty Surfacessa tai laitteesi ei herää lepotilasta, kokeile seuraavia vianmääritysohjeita:

- Jos Windows ei käynnisty oikein, kun käynnistät Surfacen (eli Windows ei käynnisty), saat ohjeet ongelman ratkaisemiseen ohjeista, jotka eivät käynnisty [Surfacella.](https://support.microsoft.com/surface/windows-doesn-t-start-on-surface-3dd47ea1-472a-4514-c8e1-ff81bd72be5c)

- Jos Surface ei käynnisty tai herää, vianmääritysohjeita varten katso, miten Surface ei käynnisty tai herää [lepotilasta.](https://support.microsoft.com/surface/surface-won-t-turn-on-or-wake-from-sleep-1e181652-3db8-5ca1-9649-7390fafb102a)

- Jos käytät kolmannen osapuolen laitetta, katso lisätietoja kolmannen osapuolen laitteiden [ongelmien vianmäärityksestä.](https://support.microsoft.com/topic/b6f3408d-dac9-43e2-82f6-e620ca783636)
