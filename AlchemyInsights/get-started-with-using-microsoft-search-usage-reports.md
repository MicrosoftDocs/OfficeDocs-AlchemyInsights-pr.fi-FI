# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Microsoft Search -käyttöraporttien käytön aloittaminen

Microsoft Search -käyttöraportit antavat tietoa siitä, miten haku toimii organisaatiossasi. Näiden tietojen avulla voit etsiä sisältöä ja tehdä haun hyödyllisempää ja miellyttävämpää käyttäjille organisaatiossasi.

Lisätietoja on microsoft search [-käyttöraporteissa.](https://go.microsoft.com/fwlink/?linkid=2152048)
