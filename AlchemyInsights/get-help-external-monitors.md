# <a name="get-help-with-external-monitors"></a>Ohjeita ulkoisten näyttöjen kanssa

Voit avata projektiasetukset helposti painamalla samanaikaisesti seuraavia näppäimiä:

- Windows-näppäin (Windows-näppäin aloituspainike) + P
- Valitse **Monista-,** **Laajenna-** tai **Toinen näyttö -vaihtoehto vain** mieltymysten mukaan.

Jos sinulla on ongelmia ulkoisten näyttöjen kanssa, kokeile seuraavia vianmääritysohjeita:

- Irrota kaapeli, joka yhdistää tietokoneen näyttöön, ja kytke se takaisin sisään.
- Irrota muut kuin tärkeät laitteet tietokoneesta (kuten sovittimet tai telakalla).

Lisätietoja Microsoft-tuesta:

- [Kahden näytön määritäminen Windows 10:ssä](https://support.microsoft.com/windows/set-up-dual-monitors-on-windows-10-3d5c15dc-cc63-d850-aeb6-b41778147554)
- [Useiden näyttöjen käyttö Windows 10:ssä](https://support.microsoft.com/windows/how-to-use-multiple-monitors-in-windows-10-329c6962-5a4d-b481-7baa-bec9671f728a)
