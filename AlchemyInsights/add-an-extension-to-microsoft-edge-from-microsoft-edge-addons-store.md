# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Laajennuksen lisääminen Microsoft Edgeen Microsoft Edge-lisä osat-kaupasta

Voit lisätä Microsoft Edgeen laajennuksen Microsoft Edge-lisä osat-kaupasta seuraavasti:

- Avaa Microsoft Edge.
- Valitse **... > laajennukset > [Hae Microsoft Edge-laajennuksia](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Valitse laajennus, jonka haluat lisätä, ja valitse sitten **Hae**.
- Tarkista vahvistus valinta ikkunassa huolellisesti laajennuksen mahdollisesti tekemät muutokset ja valitse **Lisää tunniste**.

Tämän jälkeen saat vahvistuksen siitä, että laajennus on lisätty.
