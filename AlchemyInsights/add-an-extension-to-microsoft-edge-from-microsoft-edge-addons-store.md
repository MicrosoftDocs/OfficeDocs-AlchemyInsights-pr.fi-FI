# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Laajennuksen lisääminen Microsoft Edgeen Microsoft Edgen lisäosakaupasta

Laajennuksen lisääminen Microsoft Edgeen Microsoft Edgen laajennusten kaupasta:

- Avaa Microsoft Edge.
- Valitse **... > Laajennukset > Hanki Microsoft [Edgen laajennukset.](https://go.microsoft.com/fwlink/?linkid=2136408)**
- Valitse laajennus, jonka haluat lisätä, ja valitse sitten **Hae.**
- Tarkista vahvistusvalintaikkunassa huolellisesti laajennuksen mahdollisesti tehtävät muutokset ja valitse Lisää **laajennus.**

Tämän jälkeen saat vahvistuksen siitä, että laajennus on lisätty.
