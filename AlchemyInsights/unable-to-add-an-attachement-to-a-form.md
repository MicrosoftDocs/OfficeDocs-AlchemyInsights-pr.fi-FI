# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Liitettä ei voi lisätä ulkoiseen lomakkeeseen

Jos yrität lisätä liitettä ulkoiseen Microsoft Forms-kyselyyn, ominaisuus ei ole vielä käytettävissä. Jos haluat seurattavaa ominaisuus pyynnön tilaa, siirry [tähän UserVoice-merkintään](https://go.microsoft.com/fwlink/?linkid=2133069)ja Lisää äänesi.
