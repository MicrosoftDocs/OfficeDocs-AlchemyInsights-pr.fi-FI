# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Liitettä ei voi lisätä ulkoisesti lomakkeeseen

Jos yrität lisätä liitetiedoston ulkoiseen Microsoft Forms -kyselyyn, kyseinen ominaisuus ei ole vielä käytettävissä. Jos haluat seurata kyseisen ominaisuuspyynnön tilaa, siirry [tähän UserVoice-merkintään](https://go.microsoft.com/fwlink/?linkid=2133069)ja lisää äänesi.
