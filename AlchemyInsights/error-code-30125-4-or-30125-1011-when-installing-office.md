# <a name="solutions-for-error-30125-4-or-30125-1011"></a>Virhe 30125-4 tai 30125-1011 ratkaisuja

Jos virustorjuntaohjelma, palomuuri, välityspalvelimen asetukset tai yhteyden estää sinua asentamasta Officen, näyttöön saattaa tulla virhe. Seuraavassa on joitakin asioita, voit yrittää korjata ongelman:
  
- Käynnistä tietokone uudelleen ja asenna Office uudelleen.
    
- Asenna Office käyttämällä [Office offline installer](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c.aspx)
    
- Välityspalvelimen asetusten poistaminen käytöstä tilapäisesti
    
- Virustentorjuntaohjelmiston poistaminen käytöstä tilapäisesti
    
- Palomuurin poistaminen käytöstä tilapäisesti
    
Saat lisätietoja [virhekoodin 30125-4 tai 30125-1011 Officen asennuksen yhteydessä](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612.aspx)
