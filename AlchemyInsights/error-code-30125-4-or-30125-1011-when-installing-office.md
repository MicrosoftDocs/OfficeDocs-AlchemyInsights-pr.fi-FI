# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Virhekoodi 30125-4 tai 30125-1011 Officea asennettaessa

Tämä virhe saattaa ilmetä, jos virustentorjuntaohjelmisto, palomuuri, välityspalvelimen asetukset tai yhteys estävät Officen asentamisen. Voit yrittää korjata ongelman seuraavilla asioilla:
  
- Käynnistä tietokone uudelleen ja asenna Office uudelleen

- Officen asentaminen [Officen offline-asennusohjelman](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA) avulla

- Välityspalvelimen asetusten poistaminen käytöstä tilapäisesti

- Virustentorjuntaohjelmiston poistaminen käytöstä tilapäisesti

- Palomuurin poistaminen käytöstä tilapäisesti

Lisätietoja on [virhekoodissa 30125-4 tai 30125-1011 Officea asennettaessa](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
