# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Uusi ominaisuus EOP ja ATP-vianmäärityksessä

Järjestelmänvalvojana voit nyt helposti ilmoittaa epäilyttäviä sähköpostiviestejä, URL-osoitteita ja sisällön Microsoft Explorer [Lähetykset](https://protection.office.com/reportsubmission). Sen avulla voi myös auttaa ymmärtämään, miksi lailliset viesti oli merkitty väärin. Lisätietoja tämän [ilmoituksen](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
