# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Virhe Outlook Web-tilin hankkiminen käytöstä?

Jos näyttöön tulee virhe, **tilisi on poistettu käytöstä** ja `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` kun yrität avata Outlook Web (tunnettiin aiemmin nimellä Outlook Web App), oman organisaation järjestelmänvalvoja on ehkä käytöstä Outlook Web access. Järjestelmänvalvojaa korjata tämän virheen noudattamalla ohjeen [käyttöön tai poistaa käytöstä Outlook Web Appin postilaatikkoon](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
