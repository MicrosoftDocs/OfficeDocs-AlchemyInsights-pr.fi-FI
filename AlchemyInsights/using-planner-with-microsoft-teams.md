# <a name="using-planner-with-microsoft-teams"></a>Plannerin käyttäminen Microsoft Teamsin kanssa

Voit järjestää tehtäviä lisäämällä Yhden tai useamman Planner-välilehden Teams-kanavaan: 

- [Lisää Planner-välilehti tiimin kanavalle](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_addaplannertabtoateamchannel). Välilehden lisääjän on oltava tiimin jäsen, ei vain omistaja. Voit lisätä käyttäjän jäsenenä Microsoft Teams -käyttöliittymän kautta.
- [Lisää olemassa oleva suunnitelma Microsoft Teamsissa.](https://techcommunity.microsoft.com/t5/Planner-Blog/Bringing-a-Plan-into-Microsoft-Teams/ba-p/57463)
- [Poista Planner-välilehti tai poista suunnitelma](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_removeaplannertabordeleteaplan). Poistettuja suunnitelmia ei voi palauttaa Teamsissa. Jos ryhmä on jo poistettu, voit palauttaa [poistetut suunnitelmat ja ryhmät.](https://blogs.msdn.microsoft.com/brismith/2017/03/29/microsoft-planner-now-you-can-recover-deleted-plans-and-groups)
- [Ota Plannerin tehtäväilmoitukset käyttöön tai poista ne käytöstä Teamsissa.](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_getplannerassignmentnotificationsinteams)

**Huomautus:** Saat yleiskatsauksen Plannerin ja Microsoft Teamsin integroinnista ohjeesta [Plannerin käyttö Microsoft Teamsissa.](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0)
