# <a name="what-is-microsoft-search-in-bing"></a>Mikä on Microsoft Search Bingissä?

[Microsoft Search Bingissä](https://docs.microsoft.com/deployoffice/microsoft-search-bing#what-is-microsoft-search-in-bing) on toiminto, jonka avulla voit tehdä hakuja koko organisaatiossa ja verkossa. Bingin osoitepalkin ja hakuruudun avulla voit etsiä tiedostoja, sisäisiä ja ulkoisia sivustoja, yritystyökaluja, henkilö- ja organisaatiokaavioita, tiedostoja, SharePoint-sivustoja, OneDrive-sisältöä, Yammer keskusteluja ja monia muita resursseja.
