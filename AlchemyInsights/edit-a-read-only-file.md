# <a name="edit-a-read-only-file"></a>Vain luku-tiedoston muokkaaminen

Kun napsautat tiedosto kirjastot-kohdassa tiedoston nimeä, Office-ohjelma voi odottamatta avata sen vain luku-tiedostona. Jos haluat avata sen muokkaamista varten, valitse **Avaa** (kolme pistettä) ja valitse sitten **Muokkaa.**
