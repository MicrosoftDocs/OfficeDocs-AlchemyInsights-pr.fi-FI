# <a name="edit-a-read-only-file"></a>Vain luku-tiedoston muokkaaminen

Asiakirjakirjastoissa napsauttaessasi tiedoston nimeä Office-ohjelma saattaa odottamatta avata sen vain luku-tiedostoksi. Voit avata sen muokattavaksi valitsemalla **Avaa** (kolme pistettä) ja valitse sitten **latausvaihtoehto**
