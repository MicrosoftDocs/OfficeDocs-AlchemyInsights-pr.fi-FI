# <a name="conversion-of-microsoft-365-group-and-shared-mailboxes"></a>Microsoft 365-ryhmän ja jaettujen posti laatikoiden muunto

Tällä hetkellä ei ole mahdollista muuntaa Microsoft 365-ryhmän posti laatikkoa ja jaettu posti laatikko.

Auttakaa äänestämään tätä ominaisuutta [täällä](https://aka.ms/M365GroupToShared).
