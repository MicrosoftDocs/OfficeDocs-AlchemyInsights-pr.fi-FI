# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Ongelmia Exchange-sähköpostitilin määrittämisessä Outlookissa

Jos sinulla on ongelmia Exchange-tilin määrittämisessä Outlookissa, voit suorittaa [Outlook-tilin määritystyökalun](https://aka.ms/SaRA-OutlookSetupProfile) haavoittuvuuden sisältävässä tietokoneessa. Se suorittaa erilaisia tarkistuksia ja tarjoaa mahdollisia ratkaisuja tilisi määritysongelmaan.
  
Jos ongelmat jatkuvat, lisätietoja on ohjeaiheessa [Outlook ei voi määrittää uutta profiilia Exchangen automaattisen haun avulla Exchange Online -postilaatikossa](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
