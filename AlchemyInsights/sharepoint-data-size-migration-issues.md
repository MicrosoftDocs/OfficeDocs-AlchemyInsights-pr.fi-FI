# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Ongelmia SharePoint online-tietoja siirrettäessä

Siltä, että olet siirtymässä yli 100TB tietoja SharePoint Online-sivustoon.

Valitse puhelin/email, **Uusi palvelu pyyntö** ja jättää kysely SharePoint migration yli 100 TB tekstiruutuun.

Kun lippu on lähetetty, päivitä se seuraavilla tiedoilla: 

- Yrityksen nimi ja vuokralaisen URL-osoite
- Arvioitu koko siirtymistä
- Kun haluat käynnistää ja suorittaa siirtymistä arvio
- Kuvaa, jossa siirrettävä sisältö, kuten SharePoint-palvelimeen ruutuun, GDrive, jaettuja tiedostoresursseja, jne.
