# <a name="turn-off-focused-inbox-in-email-clients"></a>Kohdistetun Saapuneet-kansion poistaminen käytöstä sähkö posti ohjelmissa

Jos haluat poistaa käytöstä tarkan Saapuneet-kansion Outlook 2016: ssa, Outlookin verkko versiossa, Outlook.com/Hotmail.com, Outlook 2016 for Macissa tai Windows 10: n sähkö postissa, tutustu Ohje aiheeseen lajitellut [Saapuneet-kansion poistaminen käytöstä](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Jos haluat poistaa käytöstä kaikkien organisaatioon kuuluvien käyttäjien lajitellut Saapuneet-kansion, tutustu Ohje aiheeseen [kaikkien organisaation käyttäjien keskitetyn kansion määrittäminen](https://docs.microsoft.com/microsoft-365/admin/setup/configure-focused-inbox).
