# <a name="configure-stay-signed-in-for-accounts"></a>Pysy kirjautuneena -tilin määrittäminen

Pysy kirjautuneena (KMSI)  näyttää Pysy kirjautuneena? -kehotteen, kun käyttäjä kirjautuu sisään onnistuneesti. Jos käyttäjä vastaa Kyllä **tähän** kehotteeseen, KMSI-palvelu antaa hänelle pysyvän päivitystunnuksen. 

Lisätietoja KMSI:n määrittämisestä on kohdassa Pysy [kirjautuneena?-kehotteen määrittäminen Azure AD -tileille.](/azure/active-directory/fundamentals/keep-me-signed-in)
