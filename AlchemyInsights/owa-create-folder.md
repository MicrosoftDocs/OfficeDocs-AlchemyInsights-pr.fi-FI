# <a name="create-a-folder"></a>Kansion luominen

1. Valitse Outlookin verkkokansion kansioruudun alareunasta **Uusi kansio**. Sinun on ehkä vieritettävä alaspäin kansioruudussa, jotta **näet Uusi kansio** -painikkeen.
1. Kirjoita kansion nimi tekstiruutuun ja paina sitten Enter-näppäintä.

Lisätietoja on [ohjeaiheessa Viestikansioiden käsitteleminen Outlookin verkkoversiossa](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
