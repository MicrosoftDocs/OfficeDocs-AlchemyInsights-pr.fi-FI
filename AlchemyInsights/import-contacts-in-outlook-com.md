# <a name="import-contacts"></a>Yhteystietojen tuominen

Tarvitset tuomalla Outlook.com yhteystiedot viedään muita email-palvelun, .csv-tiedostona. Noudata alla olevia ohjeita.
  
1. Ja sivun alareunassa Valitse **henkilöt**. 
    
2. Valikkorivillä, valitse **Hallitse** \> **Tuo yhteystiedot**. 
    
3. Valitse (vaikka .csv-tiedosto ei ole peräisin Gmail) **Gmail** koska Gmail käyttää .csv-tiedostotyyppi. 
    
4. Valitse **Selaa**, siirry kansioon, johon viedyt yhteystiedot tallennetaan ja valitse **Avaa**. 
    
Lisätietoja on kohdassa [Tuo yhteystiedot Outlook.com-tilille](https://go.microsoft.com/fwlink/p/?linkid=873136).
