# <a name="fixing-the-microsoft-365-apps-weve-run-into-a-problem-with-your-subscription-message"></a>Microsoft 365 -sovellusten korjaaminen "Olemme törmänneet tilaukseesi liittyvä ongelma" -sanoma

Jos näyttöön tulee tämä sanoma, kokeile seuraavaa:

- Jos olet vaihtanut äskettäin toiseen Microsoft 365 -tilaukseen/-palvelupakettiin, noudata [Officessa näkyviin kohdan "Tili-ilmoitus" ohjeita sen jälkeen, kun Olet vaihtanut Officen](https://support.office.com/article/account-notice-appears-in-office-after-switching-office-365-plans-857dc33a-1efc-4ce7-ac3f-ef616314e27d) uudelleen tai päivittämään Microsoft 365 -tilauksia.
- Lisätietoja [on ohjeaiheessa Office-virhe "Tili-ilmoitus: Microsoft 365 -tilauksesi ongelma on se"](https://support.office.com/article/office-error-account-notice-we-ve-run-into-a-problem-with-your-office-365-subscription-17f71ecb-f53c-4f3d-ae18-7230ca1594c1). 
- [Poista](https://docs.microsoft.com/microsoft-365/admin/manage/remove-licenses-from-users) [käyttöoikeus](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users) ja määritä se uudelleen haavoittuvuuden sisältävälle käyttäjälle.
- Avaa Office-sovellus, **File**valitse  >  **Tiedostotili**ja kirjaudu ulos kaikista tileistä. Kirjaudu uudelleen sisään käyttäjätilillä, jolla on voimassa oleva käyttöoikeus. Saat lisätietoja artikkelista [Office-tilit](https://support.office.com/article/628ea040-f265-49de-b986-be09c3ebf8a9).
- Suorita [aktivointiongelmien vianmääritystyökalu](https://aka.ms/SARA-OfficeActivation-Alchemy) ja käynnistä tietokone uudelleen.
- [Palauta Haavoittuvuuden sisältävässä tietokoneessa Officen aktivointitila](https://docs.microsoft.com/office365/troubleshoot/activation/reset-office-365-proplus-activation-state).

Katso tarkat ohjeet:
- [Näyttöön tulee tilausilmoitus, kun avaan Microsoft 365 -sovelluksen](https://support.office.com/article/4cabe32c-f594-4c0e-9191-3d3ade10cceb)
- [Officen aktivointiongelmien vianmääritys](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380)
