# <a name="blocking-items-in-your-junk-email-folder"></a>Roska posti-kansion kohteiden estäminen

Roska posti-kansiossa jo olevan kohteen estäminen poistaa sen. [Roska posti-kansion](https://outlook.live.com/mail/junkemail) kohteet poistetaan automaattisesti 30 päivän kuluttua. Valitse roska posti-kansiossa oleva kohde, jotta näet, milloin se poistetaan.

Lue lisää siitä, miten voit [estää lähettäjiä tai merkitä sähkö postin roska postiksi](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4).
