# <a name="blocking-items-in-your-junk-email-folder"></a>Roskaposti-kansion kohteiden estäminen

Roskaposti-kansiossa olevan kohteen estäminen ei poista sitä. [Roskaposti-kansion](https://outlook.live.com/mail/junkemail) kohteet poistetaan automaattisesti 30 päivän kuluttua. Valitse roskapostikansiosta kohde, niin näet, milloin se poistetaan.

Lisätietoja [lähettäjien estämisestä tai sähköpostin merkitsemisestä roskapostiksi](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4).
