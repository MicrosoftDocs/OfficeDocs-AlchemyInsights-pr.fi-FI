# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>Applen MDM Push Certificate -sertifikaatin ongelmien vianmääritys Microsoft Intunen

Ratkaise ongelma nyt alla olevien resurssien avulla.
  
1. Tässä [asiakirjassa](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get) voit määrittää tai uusia Applemdm-push-varmenteen.

2. [Intune Troubleshooting Portalin](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) avulla voit diagnosoida ja ratkaista yleisiä rekisteröintivirheitä. Lisätietoja on [tässä asiakirjassa.](https://docs.microsoft.com/intune/help-desk-operators)

3. Näissä asiakirjoissa on luettelo yleisistä virheistä, jotka estävät rekisteröitymisen ja tarkkuuden kuhunkin: [Vianmääritysopas](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) ja [Vianmääritysasiakirja](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune).
