# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>Kohteiden poistaminen tai SharePoint-sivuston roska korin tyhjentäminen 

Kun poistat kohteen SharePoint-asiakirja kirjastosta tai-luettelosta, se ei poistu heti SharePointista. Poistetut kohteet menevät sivuston roska koriin tietyn ajan. Tänä aikana voit palauttaa poistetut kohteet alkuperäiseen sijaintiin tai vapauttaa tallennus tilaa poistamalla ne.
