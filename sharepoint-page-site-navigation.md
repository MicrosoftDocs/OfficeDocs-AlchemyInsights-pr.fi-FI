# <a name="site-and-page-navigation-in-sharepoint-sites"></a>Sivuston ja sivun siirtymistä SharePoint-sivustoissa

<p>Perusperiaatteet ja hyvien käytäntöjen sivuston ja sivujen siirtymislinkit sovelletaan yhtäläisesti sekä perinteinen että Moderni SharePoint-arkkitehtuurit. <em>Toteuttaminen</em> siirtymistä varten asetusten eroaa sivustojen ja intranet puitteiden perusteella. Esimerkiksi &ldquo;peritty&rdquo; siirtymisen kokemuksia classic SharePoint-sivuston hierarkioissa (alisivustoja sisältäviä sivustoja) eivät ole käytettävissä Moderni kokemus, mutta <a href="https://support.office.com/article/fe26ae84-14b7-45b6-a6d1-948b3966427f" data-linktype="external">keskitin sivustot</a> tarjoavat hyvä tapa saavuttaa sivustojen välisen aiemmin saatavana perinteinen SharePointissa hallitun siirtymis- ja hierarkioiden siirtymistoiminnot.
